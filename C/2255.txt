           <p><em>Aprende a configurar rápidamente tu cuenta de Cloudflare y activa tu dominio. Cómo comprender las prácticas recomendadas para dominios proxy de Cloudflare.</em></p>
           <p><em>Este artículo apareció anteriormente con el título <strong>Paso 1: ¿Cómo funciona Cloudflare?</strong></em></p>
           <hr>
           <h2>Descripción</h2>
           <p>La red global de Cloudflare mejora la seguridad y el funcionamiento del sitio web. Revisa nuestro artículo sobre <a href="https://support.cloudflare.com/hc/articles/205177068">cómo funciona Cloudflare</a> para obtener más información.</p>
           <p>Activa tu dominio en Cloudflare en dos pasos:</p>
           <ol start="1">
            <li><a href="https://support.cloudflare.com/hc/articles/201720164">Crea una cuenta de Cloudflare y agrega un sitio web</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/205195708">Usa los servidores de nombres de Cloudflare en tu registrar de dominios</a></li>
           </ol>
           <div class="cf-sidebar-2 cf-note">
            El paso 2 no es obligatorio para los dominios Business y Enterprise en las <a href="https://support.cloudflare.com/hc/en-us/articles/360020348832">configuraciones de CNAME</a>.
           </div>
           <p>Además, Cloudflare recomienda varias <a href="#h_87dac948-9e12-47e0-a32c-2144aadd0a2a" target="_self">prácticas recomendadas</a> para dominios activados recientemente.&nbsp; Si<span> tienes problemas para configurar tu primer dominio de Cloudflare, consulta nuestras </span><a href="https://support.cloudflare.com/hc/articles/360027710672" target="_self">Preguntas frecuentes sobre la solución de problemas para los nuevos clientes de Cloudflare</a>.</p>
           <hr>
           <h2 id="h_87dac948-9e12-47e0-a32c-2144aadd0a2a">Prácticas recomendadas para dominios activos de Cloudflare</h2>
           <p>Cloudflare recomienda las acciones y mejoras siguientes para cada nuevo dominio que se agrega a Cloudflare:</p>
           <h3><strong>Acciones recomendadas</strong></h3>
           <p>Asegura una interacción exitosa entre tu servidor web de origen, los servicios de terceros y Cloudflare:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/201897700" target="_self">Incluye las direcciones IP de Cloudflare en la lista blanca</a> de tu servidor web de origen e</li>
            <li><a href="https://support.cloudflare.com/hc/articles/217074967">Incluye los servicios de terceros en la lista blanca</a> que usa tu sitio a través de las <a href="https://support.cloudflare.com/hc/articles/217074967" target="_self">reglas de <strong>Cloudflare IP Access</strong></a>.</li>
           </ul>
           <div class="cf-sidebar-2 cf-note">
            Las direcciones IP de tu servidor de origen se incluyen automáticamente en la lista blanca de tu dominio al agregar el dominio a Cloudflare.
           </div>
           <h3><strong>Mejoras recomendadas</strong></h3>
           <p>Conoce qué funciones de Cloudflare mejoran la velocidad de tu sitio y optimizan la seguridad:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/200172856#h_c2f90417-ee71-4b65-b6c7-3636ee2b3c63">Mejorar el rendimiento del sitio</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/115002059131">Optimizar la seguridad</a></li>
           </ul>
           <h3><strong>Lectura recomendada</strong></h3>
           <p>Asegura el tráfico a través de HTTPS, tanto para los visitantes de tu sitio como para el tráfico entre Cloudflare y tu servidor web de origen.</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/360024787372">HTTPS de extremo a extremo con Cloudflare</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/360027710672">Preguntas frecuentes sobre resolución de problemas para clientes nuevos de Cloudflare</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200172476" target="_self">Contactar a asistencia de Cloudflare</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200172676">Cómo comprender la protección DDoS de Cloudflare</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200170166" target="_self">Prácticas recomendadas: Medidas preventivas de DDoS</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/360021801151">Introducción a DNS</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/360023792171">Introducción a SSL de Cloudflare</a></li>
