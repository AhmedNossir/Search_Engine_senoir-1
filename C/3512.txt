      <p>The "<b>Related changes</b>" feature lists all recent changes (before MediaWiki 1.5: only the last) in each page linked to <i>from</i> the specified page, or all recent changes in each page with a link <i>to</i> the specified page. </p>

      <h2><span class="mw-headline" id="Details">Details</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Help:Related_changes&amp;action=edit&amp;section=1" title="Edit section: Details">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>The number of pages listed is specified in the preferences. However, the entries come from the <a href="/wiki/Special:MyLanguage/Manual:recentchanges_table" title="Special:MyLanguage/Manual:recentchanges table">recentchanges</a><span style="display:none"><a href="/wiki/Manual:Recentchanges_table" title="Manual:Recentchanges table"> </a></span> table and are therefore restricted to edits not older than <a href="/wiki/Special:MyLanguage/Manual:$wgRCMaxAge" title="Special:MyLanguage/Manual:$wgRCMaxAge">$wgRCMaxAge</a><span style="display:none"><a href="/wiki/Manual:$wgRCMaxAge" title="Manual:$wgRCMaxAge"> </a></span>. </p>
      <p>Like in <a href="/wiki/Special:MyLanguage/Help:Recent_changes" title="Special:MyLanguage/Help:Recent changes">Recent changes</a><span style="display:none"><a href="/wiki/Help:Recent_changes" title="Help:Recent changes"> </a></span> and <a href="/wiki/Special:MyLanguage/Help:New_filters_for_edit_review" title="Special:MyLanguage/Help:New filters for edit review">New filters for edit review</a><span style="display:none"><a href="/wiki/Help:New_filters_for_edit_review" title="Help:New filters for edit review"> </a></span>, <a href="/wiki/Special:MyLanguage/Help:Watching_pages" title="Special:MyLanguage/Help:Watching pages">watched pages</a> are bolded. </p> 
      <h2><span class="mw-headline" id="Watchlist-like_functionality">Watchlist-like functionality</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Help:Related_changes&amp;action=edit&amp;section=2" title="Edit section: Watchlist-like functionality">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>This feature can be used to set up custom-watchlist-like functionality. It behaves like a watchlist which would contain every page linked to from the current page. However, Related changes does not automatically include talk pages; to watch also the corresponding talk pages, links to them also have to be in the page on which Related changes is applied. </p> 
      <h2><span class="mw-headline" id="Options">Options</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Help:Related_changes&amp;action=edit&amp;section=3" title="Edit section: Options">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>The option "Hide minor edits" that can be set in the <a href="/wiki/Special:Preferences" title="Special:Preferences">preferences</a> applies among other things to Related changes. </p>
      <p>See also <a href="/wiki/Special:MyLanguage/Help:Preferences#Recent_changes" title="Special:MyLanguage/Help:Preferences">hide minor edits</a>. </p> 
      <h2><span class="mw-headline" id="Caveats">Caveats</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Help:Related_changes&amp;action=edit&amp;section=4" title="Edit section: Caveats">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>Note that for a link to redirect, only a change in the redirect is given, not a change in the target article. In this regard a direct link to the target article is superior. </p>
      <p>The "Related changes" feature neither lists changes in images or sound files nor in image description pages (regardless of which of the three linking methods (image, :image or media) is used). </p>
      <p>When applying the feature to a <a href="/wiki/Special:MyLanguage/Help:Link#Subpage_feature" title="Special:MyLanguage/Help:Link">subpage</a>, the automatic link to the parent page is ignored, i.e., changes in the parent page are not listed unless there is also an explicit link to it. </p>
