      <p>Os <a href="/wiki/Commons:Administrators" title="Commons:Administrators">administradores</a> podem proteger uma página ou imagem para restringir a edição ou a movimentação daquela página. Tal proteção pode ser indefinida ou expirar depois de um tempo específico. </p>
      <p>Essa habilidade é geralmente usada para combater <a href="/wiki/Commons:Vandalism" title="Commons:Vandalism">vandalismo</a> e e a proteção é geralmente temporária. Exceções são as predefinições pesadamente usadas e certas páginas vitais como agentes designatórios de violação de direitos autorais. </p>
      <p>O <a href="/wiki/Special:Log/protect" title="Special:Log/protect">registro de proteção</a> mostra um registro no qual as páginas foram protegidas, por quem, e a razão eles deram para a proteção. Todas as páginas que são atualmente protegidas são listadas em <a href="/wiki/Special:ProtectedPages" title="Special:ProtectedPages">Special:Protectedpages</a>. As páginas não existentes que foram protegidas contra a criação são listadas em <a href="/wiki/Special:ProtectedTitles" title="Special:ProtectedTitles">Special:Protectedtitles</a>. </p>
      <p>As páginas totalmente protegidas devem ser marcadas com a <a href="/wiki/Category:Protection_templates" title="Category:Protection templates">predefinição de proteção</a> apropriada para informar outros e auto-categorizar a página sob <a href="/wiki/Category:Protected" title="Category:Protected">Category:Protected</a>. As páginas semi-protegidas não são geralmente marcadas. </p>

      <h2><span id="Tipos_de_prote.C3.A7.C3.A3o"></span><span class="mw-headline" id="Tipos_de_proteção">Tipos de proteção</span></h2> 
      <p><span id="Types_of_protection"></span> </p> 
      <h3><span id="Prote.C3.A7.C3.A3o_total"></span><span class="mw-headline" id="Proteção_total">Proteção total</span></h3> 
      <p>A proteção total inutiliza a edição para todo mundo exceto administradores. Os arquivos de média totalmente protegidos não podem ser sobregravados por novas transferências de dados. As páginas podem ser protegidas por completo <i>indefinidamente</i> se elas forem imagens pesadamente usadas ou freqüentemente predefinições transclusas para prevenir vandalismo. A proteção total indefinida também pode ser usada para páginas que não devem ser modificadas por direitos autorais ou razões legais, como <a href="/wiki/Commons:GNU_Free_Documentation_License" title="Commons:GNU Free Documentation License">Commons:GNU Free Documentation License</a>. </p>
      <p>Páginas que experienciam guerras de edição como resultado de uma discussão podem ser <i>temporariamente</i> protegidas. Durante este tempo, os administradores só devem fazer modificações significantes se houver consenso para assim fazer. Qualquer modificação proposta deve ser discutida na página de discussão. Tais pedidos, se eles forem imediatos e incontroversos, podem ser acompanhados pela tag <a href="/wiki/Template:Editprotected" class="mw-redirect" title="Template:Editprotected"><span style="font-family:monospace,monospace;">{{Editprotected}}</span></a> para atrair a atenção dos administradores'. </p> 
      <h3><span id="Prote.C3.A7.C3.A3o_de_modelo"></span><span class="mw-headline" id="Proteção_de_modelo">Proteção de modelo</span></h3>

      <p>Template protection is used on higher risk templates such as those with a high transclusion count or those with licensing information. Templates under this protection can only be edited by administrators and those with the <a href="/wiki/Special:MyLanguage/Commons:Template_editor" title="Special:MyLanguage/Commons:Template editor">template editor</a> user right. </p> 
      <h3><span id="Semi-prote.C3.A7.C3.A3o"></span><span class="mw-headline" id="Semi-proteção">Semi-proteção</span></h3> 
      <div class="floatright">
       <a href="/wiki/File:Semi-protection-shackle.svg" class="image"><img alt="Semi-protection-shackle.svg" src="https://upload.wikimedia.org/wikipedia/commons/thumb/1/1b/Semi-protection-shackle.svg/50px-Semi-protection-shackle.svg.png" decoding="async" width="50" height="50" srcset="https://upload.wikimedia.org/wikipedia/commons/thumb/1/1b/Semi-protection-shackle.svg/75px-Semi-protection-shackle.svg.png 1.5x, https://upload.wikimedia.org/wikipedia/commons/thumb/1/1b/Semi-protection-shackle.svg/100px-Semi-protection-shackle.svg.png 2x" data-file-width="512" data-file-height="512"></a>
      </div> 
      <p>A semi-proteção inutiliza a edição de usuários anônimos e contas registradas com menos de quatro dias. Este é o tipo mais comum de proteção e é comumente usado para prevenir o vandalismo IP. </p> 
      <h3><span id="Prote.C3.A7.C3.A3o_de_movimenta.C3.A7.C3.A3o"></span><span class="mw-headline" id="Proteção_de_movimentação">Proteção de movimentação</span></h3> 
      <div class="floatright">
       <a href="/wiki/File:Move-protection-shackle.svg" class="image"><img alt="Move-protection-shackle.svg" src="https://upload.wikimedia.org/wikipedia/commons/thumb/4/44/Move-protection-shackle.svg/50px-Move-protection-shackle.svg.png" decoding="async" width="50" height="50" srcset="https://upload.wikimedia.org/wikipedia/commons/thumb/4/44/Move-protection-shackle.svg/75px-Move-protection-shackle.svg.png 1.5x, https://upload.wikimedia.org/wikipedia/commons/thumb/4/44/Move-protection-shackle.svg/100px-Move-protection-shackle.svg.png 2x" data-file-width="512" data-file-height="512"></a>
      </div> 
      <p>A proteção de movimento protege a página somente de ser movida. À revelia, as páginas totalmente protegidas são também protegidas de movimentação. </p> 
      <h3><span id="Prote.C3.A7.C3.A3o_em_cascata"></span><span class="mw-headline" id="Proteção_em_cascata">Proteção em cascata</span></h3> 
      <div class="floatright">
       <a href="/wiki/File:Cascade-protection-shackle-double-chain-link.svg" class="image"><img alt="Cascade-protection-shackle-double-chain-link.svg" src="https://upload.wikimedia.org/wikipedia/commons/thumb/8/84/Cascade-protection-shackle-double-chain-link.svg/50px-Cascade-protection-shackle-double-chain-link.svg.png" decoding="async" width="50" height="50" srcset="https://upload.wikimedia.org/wikipedia/commons/thumb/8/84/Cascade-protection-shackle-double-chain-link.svg/75px-Cascade-protection-shackle-double-chain-link.svg.png 1.5x, https://upload.wikimedia.org/wikipedia/commons/thumb/8/84/Cascade-protection-shackle-double-chain-link.svg/100px-Cascade-protection-shackle-double-chain-link.svg.png 2x" data-file-width="512" data-file-height="512"></a>
      </div> 
      <p>A proteção em cascata protege totalmente todas as páginas <a href="https://en.wikipedia.org/wiki/WP:TRANS" class="extiw" title="w:WP:TRANS">transcluídas</a> na página protegida, incluindo quaisquer imagens e sua página de descrição de arquivo e quaisquer modelos. Devido à natureza generalizada e poderosa da proteção em cascata, seu uso deve ser reduzido ao mínimo. No Commons, a proteção em cascata só deve ser usada em raras situações em que haja consenso para isso. </p>
      <p>No passado, não era possível proteger páginas inexistentes de serem criadas. Como solução alternativa, essas páginas <a href="/wiki/Commons:Protected_against_recreation" title="Commons:Protected against recreation">foram transcluídas em uma página que tinha proteção em cascata</a>. Agora é possível proteger páginas usando a proteção integrada do MediaWiki de páginas inexistentes, que é aplicada da mesma maneira que a proteção comum. No entanto, o método antigo não foi abandonado completamente. </p>
      <p>A semiproteção em cascata está <a href="https://phabricator.wikimedia.org/T10796" class="extiw" title="phab:T10796">desabilitada</a>, porque permitiu que os não-administradores protegessem totalmente as páginas, transcrevendo-as. </p> 
      <h3><span id="Prote.C3.A7.C3.A3o_de_upload"></span><span class="mw-headline" id="Proteção_de_upload">Proteção de upload</span></h3> 
      <div class="floatright">
       <a href="/wiki/File:Upload-protection-shackle.svg" class="image"><img alt="Upload-protection-shackle.svg" src="https://upload.wikimedia.org/wikipedia/commons/thumb/d/d7/Upload-protection-shackle.svg/50px-Upload-protection-shackle.svg.png" decoding="async" width="50" height="50" srcset="https://upload.wikimedia.org/wikipedia/commons/thumb/d/d7/Upload-protection-shackle.svg/75px-Upload-protection-shackle.svg.png 1.5x, https://upload.wikimedia.org/wikipedia/commons/thumb/d/d7/Upload-protection-shackle.svg/100px-Upload-protection-shackle.svg.png 2x" data-file-width="512" data-file-height="512"></a>
      </div> 
      <p>Upload protection might be used to prevent overwriting of files that are either heavily used across Wikimedia projects (e.g. template icons) or used in a dangerous location (e.g. wiki's main pages) in order to prevent vandalism. These protections might be indefinitely or temporary, for example only as long as a file is on a wiki's main page. For “heavily used” files <code>[upload=sysop]</code> <i>and</i> (to prevent malicious or accidental moves) <code>[move=sysop]</code> should be used. <code>[edit=sysop]</code> is not needed (and not useful since it, for example, does not allow category changes) if just the file's contents are important (e.g. if only very few views of the file page can be expected as it happens in non-linked template use). </p> 
      <h2><span id="Solicitando_prote.C3.A7.C3.A3o_e_desprote.C3.A7.C3.A3o"></span><span class="mw-headline" id="Solicitando_proteção_e_desproteção">Solicitando proteção e desproteção</span></h2> 
      <dl>
       <dt>
        Para pedir proteção
       </dt>
      </dl> 
      <ul>
       <li>Peça a um administrador ou faça o pedido em <a href="/wiki/Commons:Administrators%27_noticeboard/Blocks_%26_protections" class="mw-redirect" title="Commons:Administrators' noticeboard/Blocks &amp; protections">Commons:Administrators' noticeboard/Blocks &amp; protections</a>.</li>
      </ul> 
      <dl>
       <dt>
        Para pedir desproteção
       </dt>
      </dl> 
      <ul>
       <li>Primeiro, peça ao administrador que fez a proteção.</li> 
       <li>Se ele não responder, peça a um administrador diferente ou peça em <a href="/wiki/Commons:Administrators%27_noticeboard/Blocks_%26_protections" class="mw-redirect" title="Commons:Administrators' noticeboard/Blocks &amp; protections">Commons:Administrators' noticeboard/Blocks &amp; protections</a>.</li>
      </ul>
