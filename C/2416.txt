          <h2 id="2ca3W5UqLnxVQ5W5w1mm1b">Overview</h2>
          <p>The Cloudflare Dashboard manages account and domain settings. Key dashboard features include:</p>
          <ul>
           <li>Control Cloudflare features,</li>
           <li>Gain insights into domain security and caching analytics,</li>
           <li><a href="https://support.cloudflare.com/hc/articles/200167946">Share account access</a></li>
           <li>Configure settings from any device via our flexible dashboard design.</li>
          </ul>
          <p>Changes update globally on Cloudflare's network within 30 seconds.&nbsp; This applies to configuration changes for all of Cloudflare's products and features except&nbsp;<a href="https://support.cloudflare.com/hc/articles/200169246">cache purge</a>, which can take up to a minute.&nbsp;&nbsp;Changes committed via the Cloudflare API and dashboard, as well as via&nbsp;<a href="https://developers.cloudflare.com/terraform/">Terraform</a>, all utilize the same API and underlying technology.</p>
          <p>All Cloudflare plans allow&nbsp;<a href="https://support.cloudflare.com/hc/articles/200167946">sharing account access with additional users</a>. Additionally, Enterprise plans allow Administrators to create&nbsp;<a href="https://support.cloudflare.com/hc/articles/205065067">Multi-User accounts</a>&nbsp;that allow each&nbsp;user to edit only those Cloudflare settings applicable to their role.</p>
          <p></p>
          <div class="cf-sidebar-2 cf-note">
           The Cloudflare dashboard requires a web browser that supports <a href="https://caniuse.com/#search=tls%201.2">TLS 1.2</a> or newer. The Cloudflare dashboard isn’t supported on pre IE 10 browser versions. Visit <a href="https://browsehappy.com/?locale=en">Browse Happy</a> for upgrade links to the most popular browsers. 
          </div>
          <p></p>
          <p>Cloudflare uses tokens to authenticate a session when a user is logged in to their Cloudflare account. An invalid token or email error message appears if your ISP changes your IP address during your session as it invalidates the token. Log out, then log back in to correct the issue or clear your browser cache and cookies.</p>
          <hr>
          <h2 id="5bTUH7qpF3EH6ZfpLMyDgR">Summary of Cloudflare dashboard apps</h2>
          <p>The Cloudflare dashboard contains several apps focused on&nbsp;providing&nbsp;<a href="https://support.cloudflare.com/hc/en-us/articles/205075117-Understanding-the-Cloudflare-dashboard#h_76770eb2-b211-4e55-826f-8111c1d2cf30">data insights</a>,&nbsp;<a href="https://support.cloudflare.com/hc/en-us/articles/205075117-Understanding-the-Cloudflare-dashboard#h_fd2d3e1b-c379-4a41-bfb0-b643f9fd127a">optimizing security</a>, providing&nbsp;<a href="https://support.cloudflare.com/hc/en-us/articles/205075117-Understanding-the-Cloudflare-dashboard#h_6f753407-f117-4162-9bdc-0252f09884a6">configuration flexibility</a>,&nbsp;<a href="https://support.cloudflare.com/hc/en-us/articles/205075117-Understanding-the-Cloudflare-dashboard#h_7764d480-95df-4aa8-8eeb-12a9e17079e9">improving performance</a>, and enhancing site reliability.</p>
          <img src="https://images.ctfassets.net/slt3lc6tev37/17kbAxnMhT793bdHE8Ez5V/22f3d99c89bf166ae13a8b3fb9ab5afe/hc-import-dashboard_apps_no_crypto_sept19.png" alt="screenshot of the list of Cloudflare apps in the dashboard">
          <p>The information below summarizes the utility of each dashboard app.</p>
          <hr>
          <h2 id="3mMd7zkfrIf8Kw1WDTrQqX">Insights&nbsp;</h2>
          <h3>Overview</h3>
          <p><b></b></p>
          <p class="cf-video-container"> <iframe style="border:none" src="https://iframe.cloudflarestream.com/1396ff01893962ed4e4471507b428212" width="480" height="320" allowfullscreen></iframe> </p><b></b>
          <p></p>
          <p>&nbsp;Provides the following functionalities:</p>
          <ul>
           <li>Summarizes site analytics</li>
           <li>Displays notifications</li>
           <li>Links to common quick actions</li>
           <li>Summarizes plan extensions and add-on features</li>
           <li>Provides API authentication details, including <b>Zone ID </b>and <b>Account ID</b></li>
           <li>Allows changing domain plans</li>
           <li>Allows removing a site from Cloudflare</li>
          </ul>
          <h3>Analytics</h3>
          <p>Monitor statistics and trends on:</p>
          <ul>
           <li>Visitor traffic</li>
           <li>DNS request handling</li>
           <li>Threat detection</li>
           <li>Performance</li>
          </ul>
          <p></p>
          <hr>
          <h2 id="43bPquxg1NyVgmRMNTyrAj">Security</h2>
          <h3>Spectrum</h3>
          <ul>
           <li><a href="https://developers.cloudflare.com/spectrum/getting-started/">Protect all TCP/UDP ports</a> from layer 3 and 4 <a href="https://www.cloudflare.com/learning/ddos/what-is-a-ddos-attack/">Denial-of-Service attacks</a></li>
           <li>Stop traffic snooping by enabling TLS encryption for TCP services</li>
          </ul>
          <h3>SSL/TLS</h3>
          <ul>
           <li>Manage <a href="https://support.cloudflare.com/hc/en-us/articles/203295200">Cloudflare’s SSL certificate products</a></li>
           <li>Adjust <a href="https://support.cloudflare.com/hc/articles/203041594#h_812cc429-890f-421e-9c29-5fb895157522">browser supportability</a>&nbsp;by modifying TLS and encryption settings</li>
           <li><a href="https://support.cloudflare.com/hc/articles/204144518">Redirect all HTTP requests to secure HTTPS</a></li>
          </ul>
          <h3>Firewall</h3>
          <ul>
           <li><a href="https://support.cloudflare.com/hc/articles/217074967">Block or allow traffic</a> based on IP address, IP range, country, or user agent</li>
           <li><a href="https://support.cloudflare.com/hc/articles/235240767">Set rate limits</a> and restrict specific URL access to specific IPs</li>
           <li>Adjust&nbsp;<a href="https://support.cloudflare.com/hc/articles/200172016#sJbboLurEVhipzWYJQnyz">OWASP</a>&nbsp;and&nbsp;<a href="https://support.cloudflare.com/hc/articles/200172016">WAF Rule</a>&nbsp;settings </li>
           <li><a href="https://support.cloudflare.com/hc/articles/200170086">Evaluate visitor’s HTTP headers for threats</a></li>
          </ul>
          <h3><b>&nbsp;</b>Access</h3>
          <ul>
           <li><a href="https://developers.cloudflare.com/cloudflare-one">Eliminate VPNs without requiring changes at your origin</a></li>
           <li>Protect internal resources by requiring authentication</li>
           <li>Control which users and groups have access to sensitive resources</li>
           <li>Review access logs</li>
          </ul>
          <h3>ScrapeShield</h3>
          <ul>
           <li><a href="https://support.cloudflare.com/hc/articles/200170036">Protect sensitive information</a> from spammers and bots</li>
           <li><a href="https://support.cloudflare.com/hc/articles/200170016">Obfuscate email addresses</a> to prevent bot scraping</li>
           <li><a href="https://support.cloudflare.com/hc/articles/200170026">Prevent image hotlink abuse</a> from unapproved sites</li>
          </ul>
          <p></p>
          <hr>
          <h2 id="5GLUd4g91i0staGGIXS38V">Flexibility</h2>
          <h3>Workers</h3>
          <ul>
           <li><a href="https://developers.cloudflare.com/workers/about/">Run JavaScript applications in a serverless execution environment</a></li>
           <li>Scale applications without spending effort on infrastructure or operations</li>
           <li>Build granular control into requests and responses</li>
          </ul>
          <h3>Page&nbsp;Rules</h3>
          <ul>
           <li><a href="https://support.cloudflare.com/hc/articles/218411427">Provides granular control</a> for many Cloudflare settings based on a matching URL</li>
           <li><a href="https://support.cloudflare.com/hc/articles/360021023712">Cache static HTML</a><b></b></li>
          </ul>
          <h3>Custom&nbsp;Pages</h3>
          <ul>
           <li><a href="https://support.cloudflare.com/hc/articles/200172706">Create custom error pages</a> for IP blocks, WAF blocks, HTTP 5XX errors, Cloudflare HTTP 1XXX errors, and more</li>
          </ul>
          <h3>Apps</h3>
          <ul>
           <li><a href="https://support.cloudflare.com/hc/articles/115000305752">Preview and install</a> a wide range of apps that enhance your site’s security, performance, analytics, design, etc.</li>
          </ul>
          <p></p>
          <hr>
          <h2 id="24CcAO9Q1yoWO3oHlIhezV">Performance</h2>
          <h3>DNS</h3>
          <p></p>
          <p class="cf-video-container wysiwyg-text-align-center"> <iframe src="https://iframe.cloudflarestream.com/06075bdffe47898b464f65040a012a44" width="480" height="320" allowfullscreen></iframe> </p>
          <p></p>
          <ul>
           <li><a href="https://support.cloudflare.com/hc/articles/360019093151">M</a><a href="https://support.cloudflare.com/hc/articles/360019093151">anage a domain’s DNS records</a></li>
           <li><a href="https://support.cloudflare.com/hc/articles/200169626">Select which A&nbsp;or CNAME&nbsp;records to proxy</a></li>
           <li>Find the Cloudflare nameservers assigned to your domain</li>
          </ul>
          <h3>Speed </h3>
          <ul>
           <li><a href="https://support.cloudflare.com/hc/articles/219178057">Improve load time on mobile devices</a></li>
           <li><a href="https://support.cloudflare.com/hc/articles/360000607372">Compress images to accelerate page load</a></li>
           <li>Accelerate site speed by <a href="https://support.cloudflare.com/hc/articles/200168196">minifying the size of your origin source code</a></li>
           <li><a href="https://support.cloudflare.com/hc/articles/200168056">Defer loading JavaScript</a> until after your site is rendered</li>
          </ul>
          <h3>Caching</h3>
          <ul>
           <li><a href="https://support.cloudflare.com/hc/articles/200168256">Adjust caching level</a></li>
           <li><a href="https://support.cloudflare.com/hc/articles/200169246">Purge cached files</a> from all Cloudflare data centers</li>
           <li><a href="https://support.cloudflare.com/hc/articles/202775670">Tell Cloudflare what to cache</a> and <a href="https://support.cloudflare.com/hc/articles/200168276">set Browser Cache TTL</a></li>
          </ul>
          <h3>Network</h3>
          <ul>
           <li>Toggle features that enhance network performance such as&nbsp;<a href="https://www.cloudflare.com/website-optimization/http2/what-is-http2/">HTTP/2</a>&nbsp;and&nbsp;<a href="https://cloudflare-quic.com/">QUIC</a></li>
          </ul>
          <h3>Traffic</h3>
          <ul>
           <li><a href="https://www.cloudflare.com/products/argo-smart-routing/">Reduce latency and connection errors</a></li>
           <li><a href="https://developers.cloudflare.com/argo-tunnel/quickstart/">P</a><a href="https://developers.cloudflare.com/argo-tunnel/quickstart/">rotect origin web servers from IP address exposure</a>&nbsp;and attack</li>
           <li><a href="https://support.cloudflare.com/hc/articles/115000081911">Balance traffic</a>&nbsp;and prevent disruptions</li>
          </ul>
          <h3>Stream</h3>
          <ul>
           <li><a href="https://support.cloudflare.com/hc/articles/360017801091">Encode, store, and deliver your videos</a></li>
           <li>Automatically optimize format and bitrate for every device and network</li>
          </ul>
          <p></p>
