       <h3 id="about-scheduled-reminders-for-pull-requests"><a href="#about-scheduled-reminders-for-pull-requests">About scheduled reminders for pull requests</a></h3> 
       <p>Scheduled reminders help teams focus on the most important review requests that require their attention. Scheduled reminders for pull requests will send a message to your team in Slack with all open pull requests that you or your team have been asked to review, at a specified time. For example, you can create a scheduled reminder to send a message to your team's main communication channel in Slack, including all open pull requests that the team is requested to review, every Wednesday at 9:00 a.m.</p> 
       <p>Organization owners can schedule a reminder for one or more teams in their organization, for all pull requests the team or teams have been requested to review.</p> 
       <p>GitHub will only trigger reminders for up to five repositories per owner and 20 pull requests per repository.</p> 
       <h3 id="creating-a-scheduled-reminder-for-an-organization"><a href="#creating-a-scheduled-reminder-for-an-organization">Creating a scheduled reminder for an organization</a></h3> 
       <ol> 
        <li>In the top right corner of GitHub, click your profile photo, then click <strong>Your organizations</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/profile/your-organizations.png" alt="Your organizations in the profile menu"></span></li> 
        <li>Click <strong>Settings</strong> next to the organization. <span class="procedural-image-wrapper"><img src="/assets/images/help/organizations/settings-button.png" alt="The settings button"></span></li> 
        <li>In the left sidebar, click <strong>Scheduled reminders</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/organizations/scheduled-reminders-org.png" alt="Scheduled reminders button"></span></li> 
        <li>Under "Scheduled reminders", click <strong>Add your first reminder</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-add-button.png" alt="Add reminder button"></span></li> 
        <li>Under "Slack workspace", click <strong>Authorize Slack workspace</strong> and follow the instructions. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-authorize-slack.png" alt="Authorize Slack workspace button"></span></li> 
        <li>Under "Slack channel", type the name of the Slack channel where you'd like to receive notifications. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-slack-channel.png" alt="Slack channel field"></span></li> 
        <li>Under "Days", click <strong>Weekdays</strong> 
         <svg version="1.1" width="16" height="16" viewbox="0 0 16 16" class="octicon octicon-triangle-down" aria-label="The down triangle icon" role="img">
          <path d="M4.427 7.427l3.396 3.396a.25.25 0 00.354 0l3.396-3.396A.25.25 0 0011.396 7H4.604a.25.25 0 00-.177.427z"></path>
         </svg> to select one or more days you'd like to receive scheduled reminders. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-days.png" alt="Days dropdown"></span></li> 
        <li>Under "Select Time(s)", click <strong>9:00 AM</strong> 
         <svg version="1.1" width="16" height="16" viewbox="0 0 16 16" class="octicon octicon-triangle-down" aria-label="The down triangle icon" role="img">
          <path d="M4.427 7.427l3.396 3.396a.25.25 0 00.354 0l3.396-3.396A.25.25 0 0011.396 7H4.604a.25.25 0 00-.177.427z"></path>
         </svg> to select one or more times you'd like to receive scheduled reminders. Optionally, to change your timezone, click 
         <svg version="1.1" width="16" height="16" viewbox="0 0 16 16" class="octicon octicon-triangle-down" aria-label="The down triangle icon" role="img">
          <path d="M4.427 7.427l3.396 3.396a.25.25 0 00.354 0l3.396-3.396A.25.25 0 0011.396 7H4.604a.25.25 0 00-.177.427z"></path>
         </svg> next to your current timezone and select a different time zone. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-times.png" alt="Select time(s) dropdowns"></span></li> 
        <li>Under "Tracked repositories," choose which repositories you'd like the team to receive pull request review reminders for. 
         <ul> 
          <li>To receive reminders for all repositories that the chosen team has access to, click <strong>All repositories</strong>. </li> 
          <li>To receive reminders for a subset of repositories, click <strong>Only select repositories</strong>, then select one or more repositories that the chosen team has access to.</li> 
         </ul><span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-tracked-repos.png" alt="Choose tracked repositories"></span></li> 
        <li>Under "Filter by team assigned to review code", click the <strong>Add a team</strong> dropdown and choose one or more teams. You can add up to 100 teams. If the team you select doesn't have access to the "Tracked repositories" selected above, you won't be able to create the scheduled reminder. <span class="procedural-image-wrapper"><img src="/assets/images/help/organizations/scheduled-reminders-add-teams.png" alt="Add a team dropdown"></span></li> 
        <li>Optionally, to exclude draft pull requests from scheduled reminders, select <strong>Ignore drafts</strong>. For more information, see "<a href="/en/github/collaborating-with-issues-and-pull-requests/about-pull-requests#draft-pull-requests">Draft pull requests</a>." <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-ignore-drafts.png" alt="Ignore draft pull requests checkbox"></span></li> 
        <li>Optionally, to only include pull requests where a review is specifically requested from the team or a team member, select <strong>Require review requests</strong>. If you don't select this option, all pull requests are included in the scheduled reminder. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-prs-without-requests.png" alt="Ignore pull requests with no review requests checkbox"></span></li> 
        <li>Optionally, to send reminders to the pull request authors after the review requests have been fulfilled, select <strong>Remind authors after reviews</strong> and choose the number of reviews required before a reminder is sent. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-remind-authors.png" alt="Remind authors after reviews checkbox"></span></li> 
        <li>Optionally, to turn off scheduled reminders for pull requests that have already been reviewed and approved, select <strong>Ignore approved pull requests</strong>. Then, click the <strong>Ignore with 1 or more approvals</strong> to choose how many approvals a pull request must have to be ignored. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-ignore-approved-prs.png" alt="Ignore approved pull requests checkbox"></span></li> 
        <li>Under "Minimum age", type the age of a pull request, in hours. Scheduled reminders won't include pull requests that are newer than this age limit. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-min-age-field.png" alt="Ignored labels field"></span></li> 
        <li>Under "Minimum staleness", type the time since the last activity on a pull request, in hours. Scheduled reminders won't include pull requests whose last activity was more recent than this time. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-min-staleness-field.png" alt="Minimum staleness field"></span></li> 
        <li>Under "Ignored terms", type a comma-separated list of terms that may appear in titles of pull requests. Scheduled reminders won't include any pull requests that contain one or more of these terms in their titles. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-ignored-terms-field.png" alt="Ignored terms field"></span></li> 
        <li>Under "Ignored labels", type a comma-separated list of labels. Scheduled reminders won't include any pull requests that have one or more of these labels. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-ignored-labels-field.png" alt="Ignored labels field"></span></li> 
        <li>Under "Required labels", type a comma-separated list of labels. Scheduled reminders will only include pull requests that haven't already been ignored, if they have one or more of these labels. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-required-labels-field.png" alt="Required labels field"></span></li> 
        <li>Click <strong>Create reminder</strong>.</li> 
       </ol> 
       <h3 id="managing-a-scheduled-reminder-for-an-organization"><a href="#managing-a-scheduled-reminder-for-an-organization">Managing a scheduled reminder for an organization</a></h3> 
       <ol> 
        <li>In the top right corner of GitHub, click your profile photo, then click <strong>Your organizations</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/profile/your-organizations.png" alt="Your organizations in the profile menu"></span></li> 
        <li>Click <strong>Settings</strong> next to the organization. <span class="procedural-image-wrapper"><img src="/assets/images/help/organizations/settings-button.png" alt="The settings button"></span></li> 
        <li>In the left sidebar, click <strong>Scheduled reminders</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/organizations/scheduled-reminders-org.png" alt="Scheduled reminders button"></span></li> 
        <li>Next to the scheduled reminder you'd like to update, click <strong>Edit</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-edit-existing.png" alt="Edit existing scheduled reminder"></span></li> 
        <li>Make one or more changes to your scheduled reminder.</li> 
        <li>To save your changes, click <strong>Update reminder</strong>. Optionally, to test your changes while updating the changes, click <strong>Update and test reminder</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/scheduled-reminders-update-buttons.png" alt="Update buttons"></span></li> 
       </ol> 
       <h3 id="deleting-a-scheduled-reminder-for-an-organization"><a href="#deleting-a-scheduled-reminder-for-an-organization">Deleting a scheduled reminder for an organization</a></h3> 
       <ol> 
        <li>In the top right corner of GitHub, click your profile photo, then click <strong>Your organizations</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/profile/your-organizations.png" alt="Your organizations in the profile menu"></span></li> 
        <li>Click <strong>Settings</strong> next to the organization. <span class="procedural-image-wrapper"><img src="/assets/images/help/organizations/settings-button.png" alt="The settings button"></span></li> 
        <li>In the left sidebar, click <strong>Scheduled reminders</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/organizations/scheduled-reminders-org.png" alt="Scheduled reminders button"></span></li> 
        <li>
