          <h2 id="4iT66krPILQu1CZECJZr7y">Cloudflare WAF</h2>
          <p>The Cloudflare web application firewall, or WAF, is an OSI Layer 7 intelligent, integrated, and scalable solution to secure your web applications, without changing your existing infrastructure or sacrificing performance. The Cloudflare WAF protects against a large number of web attack vectors, including file inclusion, Cross-Site Scripting attacks, SQL injections, and many other vulnerabilities.</p>
          <p>This video highlights key features of the Cloudflare WAF, including how to:&nbsp;</p>
          <ul>
           <li>use the WAF rulesets to protect applications, (0:00 - 3:06)</li>
           <li>create your own Custom Firewall Rules based on your security needs, and (3:07 - 4:16)</li>
           <li>visualize and analyze threats with Firewall analytics. (4:16 - 6:11)</li>
          </ul>

          <h2 id="1X4qpsOqZadORlcwHhapHT">Cloudflare Bot Management</h2>
          <p>Cloudflare Bot Management identifies, monitors, and mitigates malicious Internet bot traffic, while still allowing useful bots to access web properties.&nbsp;</p>
          <p>Bot Management applies automated, data-driven approaches — such as heuristics, machine learning, and behavioral analysis — to mitigate issues caused by malicious bot traffic. These issues include credit card stuffing, content scraping, and spam, marketing/ad click fraud, and inventory hoarding.</p>
          <p>This video highlights key features of Cloudflare’s Bot Management, including how to:</p>
          <ul>
           <li>enable Bot Management, (0:00 - 1:55)</li>
           <li>create Firewall Rules to mitigate bot traffic, and (1:56 - 3:54)</li>
           <li>review Firewall analytics and logs for insights. (3:55 - 6:00)</li>
          </ul>

          <h2 id="5yCvDh0wAZtJIh09vuPCtj">Cloudflare Load Balanacing</h2>
          <p>Cloudflare Load Balancing improves application performance and availability by steering traffic away from unhealthy servers and dynamically distributing the traffic to the most available and responsive server pools, with no downtime.</p>
          <p>This video highlights key features of the Cloudflare’s Load Balancing, including how to:</p>
          <ul>
           <li>create a load balancer, (0:00 - 5:19)</li>
           <li>Manage health monitors and pools, and (5:20 - 6:01)&nbsp;</li>
           <li>Review load balancing analytics for insights. (6:02 - 7:49)</li>
          </ul>

          <div class="cf-sidebar-2 cf-note">

          <h2 id="7Ihgg8oCS0NKxvdhmyx97o">Cloudflare for Teams</h2>
          <p>With the Cloudflare for Teams dashboard, you can manage security policies for your people, network, and devices, all in one place.&nbsp;</p>
          <p>This dashboard demo shows you how to:</p>
          <ul>
           <li>set up Cloudflare Access, and (0:00 - 4:17)</li>
           <li>configure Cloudflare Gateway to protect your internal applications and corporate network. (4:18 - 6:57)</li>
          </ul>

          <h2 id="1S0bqs7shOjefF9yLQqucc">Cloudflare Gateway</h2>
          <p>Cloudflare for Teams uses Cloudflare Gateway to protect offices, homes, and corporate networks from malware, phishing attempts, and other security threats without sacrificing performance.</p>
          <p>Gateway provides a secure DNS resolver and filtering service that inspects and logs all DNS queries to applies policies that either block or allow the request.&nbsp;</p>
          <p>This video shows you how to get started with Cloudflare Gateway by:&nbsp;</p>
          <ul>
           <li>configuring a location, (0:00 - 2:27)</li>
           <li>creating a policy and using that policy to block security threats. (2:28 - 5:18)</li>
          </ul>
          <p></p>

           This demo was filmed using a Cloudflare domain on an Enterprise plan.

          <h2 id="5h0B1ii4g5AAnuYGjx6Oie">Cloudflare Access</h2>
          <p>Cloudflare for Teams uses Cloudflare Access to secure devices, networks, and internal applications, without compromising performance. Cloudflare Access allows you to securely expose your internal applications and services, enforce user access policies, and log per-application activity, all without a VPN.</p>
          <p>Businesses big and small use Cloudflare to protect and securely publish internally-hosted applications and infrastructure that they would traditionally protect with a network perimeter, and access via a VPN.&nbsp;</p>
          <p>This video shows you how to:&nbsp;</p>
          <ul>
           <li>enable Cloudflare Access&nbsp; (0:00 - 0:42)</li>
           <li>configure an identity provider (0:43 - 1:32)</li>
           <li>build Access policies, and (1:33 - 3:38)</li>
           <li>enable Access App Launch. (3:49 - 5:21)</li>
