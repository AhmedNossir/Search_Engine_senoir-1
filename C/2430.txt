           <h2>Descripción</h2>
           <p>El panel de control de Cloudflare gestiona la configuración de la cuenta y dominio. Las funciones clave del panel de control incluyen:</p>
           <ul>
            <li>Controlar las funciones de Cloudflare,</li>
            <li>Obtener información sobre la seguridad del dominio y el análisis de almacenamiento en caché,</li>
            <li><a href="https://support.cloudflare.com/hc/articles/200167946">Compartir el acceso de la cuenta</a> con usuarios adicionales y</li>
            <li>Configurar los ajustes desde cualquier dispositivo a través de nuestro diseño de panel de control flexible.</li>
           </ul>
           <p>Todos los planes de Cloudflare permiten <a href="https://support.cloudflare.com/hc/articles/200167946">compartir el acceso a la cuenta con usuarios adicionales</a>. Además, los Enterprise Plans permiten a los administradores crear <a href="https://support.cloudflare.com/hc/articles/205065067">Cuentas multiusuario</a> que permiten a cada usuario editar solo las configuraciones de Cloudflare que aplican a su función.</p>
           <div class="cf-sidebar-2 cf-note">
            El panel de control de Cloudflare requiere un navegador web que sea compatible con <a href="https://caniuse.com/#search=tls%201.2" target="_self">TLS 1.2</a> o posterior. El panel de control de Cloudflare no es compatible con las versiones anteriores del navegador IE 10.&nbsp;&nbsp;Visita <a href="https://browsehappy.com/?locale=en">Browse Happy</a> para ver los enlaces de actualización a los navegadores más populares.
           </div>
           <p>Cloudflare utiliza tokens para autenticar una sesión cuando un usuario inicia sesión en su cuenta de Cloudflare. Aparece un token no válido o un mensaje de error de correo electrónico si tu ISP cambia la dirección IP durante tu sesión, debido a que invalida el token. Cierra sesión, luego vuelve a iniciarla para corregir el problema o borrar el caché y las cookies de tu navegador.</p>
           <hr>
           <h2>Resumen de las aplicaciones del panel de control de Cloudflare</h2>
           <p>El panel de control de Cloudflare contiene varias aplicaciones enfocadas en proporcionar <a href="#h_76770eb2-b211-4e55-826f-8111c1d2cf30" target="_self">información de datos</a>, <a href="#h_fd2d3e1b-c379-4a41-bfb0-b643f9fd127a" target="_self">optimizar la seguridad</a>, proporcionar <a href="#h_6f753407-f117-4162-9bdc-0252f09884a6" target="_self">flexibilidad de configuración</a>, <a href="#h_7764d480-95df-4aa8-8eeb-12a9e17079e9" target="_self">mejorar el funcionamiento</a> y mejorar la confiabilidad del sitio.</p>
           <p><img alt="dashboard_apps_no_crypto_Sept19.png" src="https://support.cloudflare.com/hc/article_attachments/360038338392/dashboard_apps_no_crypto_Sept19.png"></p>
           <p>La información a continuación resume la utilidad de cada aplicación del panel de control:</p>
           <h3 id="h_76770eb2-b211-4e55-826f-8111c1d2cf30">Información&nbsp;</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;Descripción</strong></p>
           <p class="cf-video-container"><iframe allowfullscreen height="320" src="https://iframe.cloudflarestream.com/1396ff01893962ed4e4471507b428212" style="border: none;" width="480"></iframe></p>
           <p>&nbsp; &nbsp; &nbsp;Proporciona las siguientes funcionalidades:</p>
           <ul>
            <li>Resume el análisis del sitio</li>
            <li>Muestra las notificaciones</li>
            <li>Enlaza a acciones rápidas comunes</li>
            <li>Resume las extensiones del plan y funciones adicionales</li>
            <li>Proporciona detalles de autenticación de API, incluidas la <strong>identificación de la zona</strong> y la <strong>identificación de la cuenta</strong></li>
            <li>Permite cambiar los planes de dominio</li>
            <li>Permite eliminar un sitio de Cloudflare&nbsp;</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Análisis</strong></p>
           <p>&nbsp; &nbsp; &nbsp;Monitorea las estadísticas y tendencias sobre:</p>
           <ul>
            <li>El tráfico del visitante</li>
            <li>Manejo de solicitudes de DNS</li>
            <li>Detección de amenazas</li>
            <li>Funcionamiento</li>
           </ul>
           <h3 id="h_fd2d3e1b-c379-4a41-bfb0-b643f9fd127a">Seguridad</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;Spectrum</strong>:</p>
           <ul>
            <li><a href="https://developers.cloudflare.com/spectrum/getting-started/">Protege todos los puertos TCP/UDP</a> de los <a href="https://www.cloudflare.com/learning/ddos/what-is-a-ddos-attack/">ataques de denegación de servicio</a> de capa 3 y 4</li>
            <li>Detiene la intrusión de tráfico al habilitar el cifrado TLS para los servicios TCP</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;SSL/TLS</strong>:</p>
           <ul>
            <li>Gestiona los <a href="https://support.cloudflare.com/hc/en-us/articles/203295200" target="_self">productos de certificado SSL de Cloudflare</a></li>
            <li>Ajusta la <a href="https://support.cloudflare.com/hc/articles/203041594#h_812cc429-890f-421e-9c29-5fb895157522">compatibilidad del navegador</a> al modificar el TLS y la configuración de cifrado</li>
            <li><a href="https://support.cloudflare.com/hc/articles/204144518#h_a61bfdef-08dd-40f8-8888-7edd8e40d156">Redirecciona todas las solicitudes de HTTP para asegurar el HTTPS</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Firewall</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/217074967">Bloquea o habilita el tráfico</a> en función de la dirección IP, rango de IP, país o agente de usuario</li>
            <li><a href="https://support.cloudflare.com/hc/articles/235240767">Establece límites de velocidad</a> y restringe el acceso de URL específico a IP específicas</li>
            <li>Ajusta la configuración de las reglas <a href="https://support.cloudflare.com/hc/articles/360002866492">OWASP</a> y <a href="https://support.cloudflare.com/hc/articles/200172016">WAF</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200170086">Evalúa los encabezados HTTP del visitante para detectar amenazas</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Access</strong>:</p>
           <ul>
            <li><a href="https://developers.cloudflare.com/access/about/">Elimina las VPN sin necesidad de hacer cambios en tu origen</a></li>
            <li>Protege los recursos internos al requerir autenticación</li>
            <li>Controla qué usuarios y grupos tienen acceso a los recursos confidenciales</li>
            <li>Revisa los registros de acceso</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;ScrapeShield</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/200170036">Protege la información confidencial</a> de los spammers y bots</li>
            <li><a href="https://support.cloudflare.com/hc/articles/200170016">Ofusca las direcciones de correo electrónico</a> para prevenir el bot scrapping</li>
            <li><a href="https://support.cloudflare.com/hc/articles/200170026">Previene el abuso de enlaces de imagen</a> de sitios no aprobados</li>
           </ul>
           <h3 id="h_6f753407-f117-4162-9bdc-0252f09884a6">Flexibilidad</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;Workers</strong>:</p>
           <ul>
            <li><a href="https://developers.cloudflare.com/workers/about/">Ejecuta aplicaciones JavaScript en un entorno de ejecución sin servidor</a></li>
            <li><span>Escala las aplicaciones sin invertir esfuerzos en infraestructura u operaciones</span></li>
            <li><span>Crea un control más detallado de las solicitudes y respuestas</span></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Page</strong> <strong>Rules</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/218411427">Proporciona control más detallado</a> de muchas configuraciones de Cloudflare basadas en una URL combinable</li>
            <li><a href="https://support.cloudflare.com/hc/articles/360021023712">Caché HTML estático</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Páginas</strong> <strong>personalizadas</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/200172706">Crea páginas de error personalizadas</a> para bloques de IP, bloques WAF, errores HTTP 5XX, errores Cloudflare HTTP 1XXX y más</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Aplicaciones</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/115000305752">Previsualiza e instala</a> una amplia gama de aplicaciones que mejoran la seguridad, el funcionamiento, el análisis, el diseño, etc. de tu sitio.</li>
           </ul>
           <h3 id="h_7764d480-95df-4aa8-8eeb-12a9e17079e9">Funcionamiento</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;DNS</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/360019093151">Gestiona</a><a href="https://support.cloudflare.com/hc/articles/360019093151"> los registros DNS de un dominio</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200169626" target="_self">Selecciona qué registros A o CNAME se redirigen mediante proxy</a> a Cloudflare</li>
            <li>Encuentra los servidores de nombres de Cloudflare asignados a tu dominio</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Speed</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/219178057">Mejora el tiempo de carga en dispositivos móviles</a> que usan redes lentas</li>
            <li><a href="https://support.cloudflare.com/hc/articles/360000607372">Comprime imágenes para acelerar la carga de la página</a></li>
            <li>Acelera la velocidad del sitio al <a href="https://support.cloudflare.com/hc/articles/200168196" target="_self">minimizar el tamaño de tu código fuente de origen</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200168056">Aplaza la carga de JavaScript</a> hasta que se haya procesado tu sitio</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Caching</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/200168256">Ajusta el nivel de almacenamiento en caché</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200169246">Purga los archivos en caché</a> de todos los centros de datos de Cloudflare</li>
            <li><a href="https://support.cloudflare.com/hc/articles/202775670">Le indica a Cloudflare qué almacenar en caché</a> y <a href="https://support.cloudflare.com/hc/articles/200168276">configura el browser cache expiration</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Red</strong>:</p>
           <ul>
            <li>Alterna las funciones que mejoran el funcionamiento de la red, como <a href="https://www.cloudflare.com/website-optimization/http2/what-is-http2/">HTTP/2</a> y <a href="https://cloudflare-quic.com/">QUIC</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Tráfico</strong>:</p>
           <ul>
            <li><a href="https://www.cloudflare.com/products/argo-smart-routing/">Reduce la latencia y los errores de conexión</a></li>
            <li><a href="https://developers.cloudflare.com/argo-tunnel/quickstart/">P</a><a href="https://developers.cloudflare.com/argo-tunnel/quickstart/">rotege los servidores web de origen de ataques y exposición</a> de la dirección IP</li>
            <li><a href="https://support.cloudflare.com/hc/articles/115000081911">Establece un equilibrio en el tráfico</a> y evita interrupciones</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Stream</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/360017801091">Codifica, almacena y entrega tus videos</a></li>
            <li>Optimiza automáticamente el formato y bitrate para cada dispositivo y red</li>
           </ul>
