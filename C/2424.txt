           <p><em>Hier lernen Sie die Apps des Cloudflare Dashboards kennen, die für Ihre Bedürfnisse am relevantesten sind.</em></p>
           <hr>
           <h2>Übersicht</h2>
           <p>Im Cloudflare Dashboard werden Konto- und Domain-Einstellungen verwaltet. Die wichtigsten Features des Dashboards sind:</p>
           <ul>
            <li>Steuerung von Cloudflare-Features,</li>
            <li>Informationen über Domain-Sicherheit und Caching-Analysen erhalten,</li>
            <li>Weiteren Benutzern <a href="https://support.cloudflare.com/hc/articles/200167946">Kontozugang gewähren</a> und</li>
            <li>Einstellungen von jedem Gerät aus konfigurieren. Unser Dashboard ist dazu besonders flexibel gestaltet.</li>
           </ul>
           <p>Bei allen Cloudflare-Tarifen können Sie <a href="https://support.cloudflare.com/hc/articles/200167946">weiteren Benutzern Kontozugang gewähren</a>. Beim Enterprise Plan kann der Administrator außerdem <a href="https://support.cloudflare.com/hc/articles/205065067">Multi-User-Konten</a> erstellen. Dabei kann jeder Benutzer nur die Cloudflare-Einstellungen bearbeiten, die für seine Rolle gelten.</p>
           <div class="cf-sidebar-2 cf-note">
            Für das Cloudflare Dashboard ist ein Webbrowser erforderlich, der <a href="https://caniuse.com/#search=tls%201.2" target="_self">TLS 1.2</a> oder neuer unterstützt. Auf Browserversionen vor IE 10 wird das Cloudflare Dashboard nicht unterstützt.&nbsp;&nbsp;Auf <a href="https://browsehappy.com/?locale=en">Browse Happy</a> finden Sie Upgrade-Links der beliebtesten Browser.
           </div>
           <p>Cloudflare verwendet Tokens, um eine Sitzung zu authentifizieren, wenn ein Benutzer bei seinem Cloudflare-Konto angemeldet ist. Wenn Ihr ISP während der Sitzung Ihre IP-Adresse ändert, erscheint eine Fehlermeldung „Token oder E-Mail ungültig“, denn das Token wird dadurch ungültig. Melden Sie sich ab und dann wieder an, um das Problem zu beheben, oder löschen Sie den Cache und die Cookies Ihres Browsers.</p>
           <hr>
           <h2>Übersicht über die Apps im Cloudflare Dashboard</h2>
           <p>Das Cloudflare Dashboard enthält verschiedene Apps mit den Schwerpunkten <a href="#h_76770eb2-b211-4e55-826f-8111c1d2cf30" target="_self">Datenerkenntnisse</a>,&nbsp;<a href="#h_fd2d3e1b-c379-4a41-bfb0-b643f9fd127a" target="_self">Sicherheitsoptimierung</a>, <a href="#h_6f753407-f117-4162-9bdc-0252f09884a6" target="_self">flexible Konfiguration</a>,&nbsp;<a href="#h_7764d480-95df-4aa8-8eeb-12a9e17079e9" target="_self">Leistungssteigerung</a> und Zuverlässigkeit der Website.</p>
           <p><img alt="dashboard_apps_no_crypto_Sept19.png" src="https://support.cloudflare.com/hc/article_attachments/360038338392/dashboard_apps_no_crypto_Sept19.png"></p>
           <p>Unten finden Sie Informationen über den Zweck jeder Dashboard-App:</p>
           <h3 id="h_76770eb2-b211-4e55-826f-8111c1d2cf30">Erkenntnisse&nbsp;</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;Überblick</strong></p>
           <p class="cf-video-container"><iframe allowfullscreen height="320" src="https://iframe.cloudflarestream.com/1396ff01893962ed4e4471507b428212" style="border: none;" width="480"></iframe></p>
           <p>&nbsp; &nbsp; &nbsp;Enthält folgende Funktionen:</p>
           <ul>
            <li>Zusammenfassung von Website-Analytics</li>
            <li>Anzeige von Benachrichtigungen</li>
            <li>Links zu häufig benötigten schnellen Maßnahmen</li>
            <li>Zusammenfassung von Tariferweiterungen und zusätzlichen Features</li>
            <li>Details zur API-Authentifizierung einschließlich <strong>Zonen-ID</strong> und <strong>Konto-ID</strong></li>
            <li>Ändern von Domain-Tarifen</li>
            <li>Entfernen einer Website aus Cloudflare&nbsp;</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Analytics</strong></p>
           <p>&nbsp; &nbsp; &nbsp;Beobachtung von Statistiken und Trends zu:</p>
           <ul>
            <li>Besucher-Traffic</li>
            <li>DNS-Anfragebehandlung</li>
            <li>Bedrohungserkennung</li>
            <li>Performance</li>
           </ul>
           <h3 id="h_fd2d3e1b-c379-4a41-bfb0-b643f9fd127a">Sicherheit</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;Spectrum</strong>:</p>
           <ul>
            <li><a href="https://developers.cloudflare.com/spectrum/getting-started/">Schutz aller TCP/UDP-Ports</a>&nbsp;vor <a href="https://www.cloudflare.com/learning/ddos/what-is-a-ddos-attack/">Denial-of-Service-Angriffen</a> auf Layer 3 und 4</li>
            <li>Schutz vor Ausspähen von Traffic durch TLS-Verschlüsselung für TCP-Dienste</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;SSL/TLS</strong>:</p>
           <ul>
            <li>Verwaltung der <a href="https://support.cloudflare.com/hc/en-us/articles/203295200" target="_self">Cloudflare-SSL-Zertifikatprodukte</a></li>
            <li>Anpassung der <a href="https://support.cloudflare.com/hc/articles/203041594#h_812cc429-890f-421e-9c29-5fb895157522">Browser-Unterstützung</a>&nbsp;durch Änderung von TLS- und Verschlüsselungseinstellungen</li>
            <li><a href="https://support.cloudflare.com/hc/articles/204144518#h_a61bfdef-08dd-40f8-8888-7edd8e40d156">Umleitung aller HTTP-Anfragen zu sicherem HTTPS</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Firewall</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/217074967">Blockieren oder Zulassen von Traffic</a>&nbsp;je nach IP-Adresse, IP-Bereich, Land oder User-Agent</li>
            <li><a href="https://support.cloudflare.com/hc/articles/235240767">Festlegung der Ratenbegrenzung</a>&nbsp;und Einschränkung des Zugriffs auf bestimmte URLs auf bestimmte IPs</li>
            <li>Anpassung von <a href="https://support.cloudflare.com/hc/articles/360002866492">OWASP</a>&nbsp;und <a href="https://support.cloudflare.com/hc/articles/200172016">WAF-Regeln</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200170086">Untersuchung der HTTP-Header von Besuchern auf Bedrohungen</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Access</strong>:</p>
           <ul>
            <li><a href="https://developers.cloudflare.com/access/about/">VPN-Eliminierung, ohne dass etwas am Ursprungsserver geändert werden muss</a></li>
            <li>Schutz interner Ressourcen durch Anforderung einer Authentifizierung</li>
            <li>Steuerung, welche Benutzer und Gruppen Zugriff auf sensible Ressourcen bekommen</li>
            <li>Aufruf der Access-Protokolle</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;ScrapeShield</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/200170036">Schutz sensibler Informationen</a>&nbsp;vor Spammern und Bots</li>
            <li><a href="https://support.cloudflare.com/hc/articles/200170016">Verschleierung von E-Mail-Adressen</a>&nbsp;zur Abwehr von Bot-Scraping</li>
            <li><a href="https://support.cloudflare.com/hc/articles/200170026">Verhindern von Bild-Hotlink-Missbrauch</a>&nbsp;durch unbefugte Websites</li>
           </ul>
           <h3 id="h_6f753407-f117-4162-9bdc-0252f09884a6">Flexibilität</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;Workers</strong>:</p>
           <ul>
            <li><a href="https://developers.cloudflare.com/workers/about/">Ausführen von JavaScript-Anwendungen in einer serverlosen Ausführungsumgebung</a></li>
            <li><span>Skalierung von Anwendungen ohne Aufwand für Infrastruktur oder operativen Betrieb</span></li>
            <li><span>Granulare Kontrolle innerhalb von Anfragen und Antworten</span></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Page</strong> <strong>Rules</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/218411427">Granulare Kontrolle</a>&nbsp;für viele Cloudflare-Einstellungen auf der Grundlage einer passenden URL</li>
            <li><a href="https://support.cloudflare.com/hc/articles/360021023712">Speichern von statischem HTML im Cache</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Benutzerdefinierte</strong> <strong>Seiten</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/200172706">Erstellung benutzerdefinierter Fehlerseiten</a>&nbsp;für IP-Sperren, WAF-Sperren, „HTTP 5XX“-Fehler, „Cloudflare HTTP 1XXX“-Fehler und mehr</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Apps</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/115000305752">Vorschau und Installation</a>&nbsp;einer breiten Palette von Apps für mehr Sicherheit und bessere Performance Ihrer Webseite, bessere Analytics, besseres Design etc.</li>
           </ul>
           <h3 id="h_7764d480-95df-4aa8-8eeb-12a9e17079e9">Performance</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;DNS</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/360019093151">V</a><a href="https://support.cloudflare.com/hc/articles/360019093151">erwaltung der DNS-Einträge einer Domain</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200169626" target="_self">Auswahl der A- und CNAME-Einträge</a>, die über den Cloudflare-Proxy geleitet werden</li>
            <li>Heraussuchen der Cloudflare-Nameserver, die Ihrer Domain zugeordnet sind</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Speed</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/219178057">Verbesserung der Ladezeiten bei Mobilgeräten</a>, die mit langsamen Netzwerken arbeiten</li>
            <li><a href="https://support.cloudflare.com/hc/articles/360000607372">Bildkomprimierung zur Beschleunigung des Seitenaufrufs</a></li>
            <li>Verbesserung der Seitengeschwindigkeit durch <a href="https://support.cloudflare.com/hc/articles/200168196" target="_self">Minimierung der Größe des Ursprungs-Quellcodes</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200168056">Verzögertes Laden von JavaScript</a>&nbsp;bis nach dem Seitenaufbau</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Caching</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/200168256">Anpassen des Cache-Levels</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200169246">Löschen zwischengespeicherter Dateien</a> auf allen Cloudflare-Rechenzentren</li>
            <li><a href="https://support.cloudflare.com/hc/articles/202775670">Einstellung, was bei Cloudflare zwischengespeichert werden soll</a> und <a href="https://support.cloudflare.com/hc/articles/200168276">Festlegung der Browser Cache Expiration</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Network</strong>:</p>
           <ul>
            <li>Ein- und Ausschalten von Features, die die Netzwerkleistung verbessern, zum Beispiel <a href="https://www.cloudflare.com/website-optimization/http2/what-is-http2/">HTTP/2</a>&nbsp;und <a href="https://cloudflare-quic.com/">QUIC</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Traffic</strong>:</p>
           <ul>
            <li><a href="https://www.cloudflare.com/products/argo-smart-routing/">Verringerung von Latenz und Verbindungsfehlern</a></li>
            <li><a href="https://developers.cloudflare.com/argo-tunnel/quickstart/">S</a><a href="https://developers.cloudflare.com/argo-tunnel/quickstart/">chutz der Ursprungswebserver vor der Offenlegung ihrer IP-Adresse</a>&nbsp;und vor Angriffen</li>
            <li><a href="https://support.cloudflare.com/hc/articles/115000081911">Ausgleich von Traffic</a>&nbsp;und Verhindern von Störungen</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Stream</strong>:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/360017801091">Codierung, Speicherung und Bereitstellung Ihrer Videos</a></li>
            <li>Automatische Optimierung von Format und Bitrate für jedes Gerät und Netzwerk</li>
           </ul>
