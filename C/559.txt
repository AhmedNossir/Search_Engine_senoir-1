      <p>Una <b>categoria</b> és un recurs especial del programari <a href="https://en.wikipedia.org/wiki/MediaWiki" class="extiw" title="w:MediaWiki">MediaWiki</a>, una pàgina especial que s'usa per agrupar pàgines i fitxers multimèdia relacionats. A la pràctica, implica que associeu un tema únic a una categoria determinada. El nom de la categoria seria suficient per endevinar el tema, però algun text addicional pot ser útil per definir-lo amb precisió. L'estructura de categories és la forma principal d'organitzar i trobar fitxers a Commons. És essencial que tots els fitxers es puguin trobar navegant per l'estructura de categories. Per permetre-ho, cada fitxer s'ha de posar directament en una categoria. Cada categoria hauria d'estar ella mateixa en categories més generals, formant una estructura jeràrquica. </p>

      <h2><span class="mw-headline" id="Guia_breu">Guia breu</span></h2> 
      <p>1. Com trobar les categories adequades </p> 
      <ul>
       <li>Cerqueu categories amb el motor de cerca (vegeu <a href="#Consells_de_categorització">#Consells de categorització</a>)</li> 
       <li>o comproveu com es classifiquen fitxers similars (és possible que alguns no estiguin ben classificats)</li> 
       <li>o bé comenceu des de la categoria arrel principal (<a href="/wiki/Category:Topics" title="Category:Topics">Category:Topics</a>)</li> 
       <li>A partir d'aquestes categories, comproveu els pares de la categoria on esteu o be les seves subcategories per trobar una categoria adequada. Eviteu escollir categories massa generals.</li>
      </ul> 
      <p>2. Afegiu les categories al fitxer </p> 
      <ul>
       <li>Seleccioneu-los al formulari de pujada</li> 
       <li>o bé: afegiu-los manualment afegint, p. ex. <code>[[Category:Foobar]]</code> a la pàgina de descripció del fitxer (vegeu <a href="#Categorització_de_les_pàgines">#Categorització de les pàgines</a>)</li> 
       <li>o bé: utilitzeu HotCat (consulteu <a href="/wiki/Special:MyLanguage/Help:Gadget-HotCat" title="Special:MyLanguage/Help:Gadget-HotCat">Help:Gadget-HotCat</a>)</li> 
       <li>o bé: utilitzeu Cat-a-lot (consulteu <a href="/wiki/Special:MyLanguage/Help:Gadget-Cat-a-lot" title="Special:MyLanguage/Help:Gadget-Cat-a-lot">Help:Gadget-Cat-a-lot</a>)</li>
      </ul>

      <p>Els principis principals són: </p> 
      <h4><span class="mw-headline" id="Principi_de_jerarquia">Principi de jerarquia</span></h4> 
      <p>L'estructura de categories és (idealment) una <a href="https://en.wikipedia.org/wiki/ca:Graf_ac%C3%ADclic_dirigit" class="extiw" title="w:ca:Graf acíclic dirigit">multi-jerarquia</a> amb una única categoria arrel, <a href="/wiki/Category:CommonsRoot" title="Category:CommonsRoot">Category:CommonsRoot</a>. Totes les categories (excepte la categoria arrel <a href="/wiki/Category:CommonsRoot" title="Category:CommonsRoot"><i>CommonsRoot</i></a>) han d'estar contingudes en una altra o altres categories, i no hauria d'haver bucles (és a dir, que una categoria es contingui a si mateixa, directament o indirectament). </p> 
      <h4><span class="mw-headline" id="Principi_de_modularitat">Principi de modularitat</span></h4> 
      <p>La pàgina (fitxer, categoria) s'hauria de situar en la categoria o categories més específiques que s'adaptin a la pàgina (no directament a les seves categories principals). Una categoria pot tenir més categories parentals. Una categoria pot combinar dos (o més) criteris diferents; aquestes categories s'anomenen "categories compostes" o "categories d'intersecció". Per exemple. la categoria d'arrel <a href="/wiki/Category:Churches" title="Category:Churches">Category:Churches</a> i la categoria d'arrel <a href="/wiki/Category:Russia" title="Category:Russia">Category:Russia</a> tenen una subcategoria comuna <a href="/wiki/Category:Churches_in_Russia" title="Category:Churches in Russia">Churches in Russia</a>. </p> 
      <h4><span class="mw-headline" id="Principi_de_simplicitat">Principi de simplicitat</span></h4> 
      <p>Aquest principi suggereix no combinar massa criteris diferents. </p> 
      <h4><span class="mw-headline" id="Principi_de_selectivitat">Principi de selectivitat</span></h4> 
      <p>No hem de classificar els elements relacionats amb temes diferents de la mateixa categoria. El nom de la categoria ha de ser inequívoc i no homònim. </p> 
      <h4><span id="Principi_d.E2.80.99universalitat"></span><span class="mw-headline" id="Principi_d’universalitat">Principi d’universalitat</span></h4> 
      <p>Els elements idèntics haurien de tenir noms idèntics per a tots els països i en tots els nivells de categorització. L'estructura de classificació hauria de ser el més sistemàtica i unificada possible, els dialectes locals i la terminologia haurien de suprimir-se a favor de la universalitat, si és possible. Les branques de categorització analògica haurien de tenir una estructura analògica. </p> 
      <h3><span class="mw-headline" id="Els_tipus_de_les_relacions_reflectides">Els tipus de les relacions reflectides</span></h3> 
      <p>L’estructura de categories ha de reflectir una jerarquia de conceptes, del més genèric al més específic. L’estructura utilitza i combina més tipus de relació, per exemple: </p> 
      <ul>
       <li><a href="https://en.wikipedia.org/wiki/en:Hyponymy_and_hypernymy" class="extiw" title="w:en:Hyponymy and hypernymy">Hiponímia</a>: una classe o tipus de… (típicament en taxonomia biològica)</li> 
       <li><a href="https://en.wikipedia.org/wiki/ca:Meron%C3%ADmia" class="extiw" title="w:ca:Meronímia">Meronímia</a>: una part de…, un membre de… (típicament per a divisions geogràfiques o relacions com a edifici/habitació, dispositiu/component, entre d’altres).</li> 
       <li>Atributs: 
        <ul>
         <li>Atributs qualitatius i generals (color, forma, mida, capacitat o discapacitat, nacionalitat, tècnica, qualitat, premis ...)</li> 
         <li>Ubicació: on, en..., des de... (lloc/esdeveniment, lloc/edifici, lloc/exposició, lloc/gent, país/idioma, font/treball, fàbrica o país/producte, etc.)</li> 
         <li>Temps: quan (hora/esdeveniment, hora/situació representada, hora de naixement, inici o construcció, hora de mort, demolició o finalització, etc.)</li>
        </ul></li> 
       <li>Relacions d'agència i d'influència: (creador/treball, dispositiu/producte, empresa/producte, disciplina o professió/els seus subjectes i termes, pares/fills, subordinació, propietari/propietat, iniciador/seguidor, subjecte/altres subjectes dedicats a allò o en el seu nom després d'allò, assumpte/el seu duplicat, imitació, representació o símbol, etc., etc.)</li> 
       <li>Modificació: original/modificat o modificat/original (eviteu l'estructura cíclica): subjectes canviats de nom, reconstruïts, reutilitzats, repensats o transformats.</li>
      </ul> 
      <h3><span class="mw-headline" id="Categories_principals">Categories principals</span></h3> 
      <p>Les categories més altes (les contingudes directament en CommonsRoot) divideixen l'estructura de les categories segons el propòsit de les altres categories que continguin. </p> 
      <ul>
       <li><b><a href="/wiki/Category:Topics" title="Category:Topics">Category:Topics</a></b> - Aquesta categoria és l'arrel comuna global dels fitxers multimèdia categoritzats segons el tema o assumpte que il·lustren. Tots els fitxers de mitjans haurien categoritzar sota aquesta categoria per permetre a altres usuaris trobar en buscar per tema.</li> 
       <li><b><a href="/wiki/Category:Copyright_statuses" title="Category:Copyright statuses">Category:Copyright statuses</a></b> - Aquesta categoria és l'arrel comuna global dels fitxers multimèdia categoritzats segons la seva llicència (per la qual s'allibera el copyright o dret de reproducció). Tots els fitxers de mitjans haurien categoritzar sota aquesta categoria amb la marca de llicència adequada.</li> 
       <li><b><a href="/wiki/Category:Image_sources" title="Category:Image sources">Category:Image sources</a></b> - Aquesta categoria és l'arrel comuna global dels fitxers multimèdia categoritzats segons l'origen o font de la qual provenen (llibres, col·leccions, llocs web, etc.)</li> 
       <li><b><a href="/wiki/Category:Media_types" title="Category:Media types">Category:Media types</a></b> - Aquesta categoria és l'arrel comuna global dels fitxers multimèdia categoritzats segons el tipus de mitjà (fitxers d'imatge fixa, vídeo, sons, etc.). Freqüentment s'omet aquest tipus de categorització amb les imatges, ja que la gran majoria de fitxers en Commons són algun tipus d'imatges.</li> 
       <li><b><a href="/wiki/Category:Commons" title="Category:Commons">Category:Commons</a></b> - Aquesta categoria és l'arrel comuna global per categoritzar les tasques de manteniment de Commons i les pàgines (Commons:-, and Help:-), excepte els fitxers multimèdia.</li>
      </ul> 
      <p>Les pàgines traduïdes han categoritzar-se en la categoria pròpia de cada llengua amb l'estil "Category:Commons-<a href="https://en.wikipedia.org/wiki/ISO_639" class="extiw" title="w:ISO 639">ISO-LANGUAGE-CODE</a>". L'estructura de <a href="/wiki/Category:Commons-en" title="Category:Commons-en">Category:Commons-en</a> és la jerarquia de mostra per a qualsevol altra subcategoria lingüística. No utilitzeu dos punts als noms de categories o pàgines. Vegeu <a href="/wiki/Category_talk:Commons_batch_uploading" title="Category talk:Commons batch uploading">aquesta discussió</a> i <a href="/wiki/Special:MyLanguage/Help:Namespaces" title="Special:MyLanguage/Help:Namespaces">Help:Namespaces</a>. </p> 
      <dl>
       <dd>
        <dl>
         <dd>
          Hi ha la subcategoria <a href="/wiki/Category:Commons_maintenance_content" title="Category:Commons maintenance content">Category:Commons maintenance content</a>, que és per al manteniment especial de continguts comuns globals de Wikimedia Commons i que no es tradueix. TOTS els fitxers multimèdia s'han de classificar a les 4 primeres categories següents, però NOMÉS els fitxers que tinguin problemes i que s'hagin de solucionar també haurien de pertànyer a la subcategoria <a href="/wiki/Category:Commons_maintenance_content" title="Category:Commons maintenance content">Category:Commons maintenance content</a>.
         </dd>
        </dl>
       </dd>
      </dl> 
      <ul>
       <li><b><a href="/wiki/Category:User_categories" title="Category:User categories">Category:User categories</a></b> - es tracta de categories que contenen galeries, imatges i textos dels usuaris de Commons, ordenats per coses com l'idioma que parlen. També conté <a href="/wiki/Category:User_galleries" title="Category:User galleries">Category:User galleries</a>, que és per a galeries específiques de l'usuari (és a dir, no temàtiques) que no necessiten estar en anglès.</li>
      </ul> 
      <h2><span id=".C3.9As_de_les_categories"></span><span class="mw-headline" id="Ús_de_les_categories">Ús de les categories</span></h2> 
      <p>Hauríeu de posar els fitxers que carregueu en categories i/o pàgines de galeries d'acord amb el seu tema, perquè altres puguin trobar i utilitzar les vostres contribucions. </p>
      <p>No sol ser necessari crear una categoria nova (hi ha excepcions, com ara penjar un text nou i veure <i><a href="#Persones">Persones</a></i> a continuació)- abans de fer-ho, assegura't d'estar familiaritzat amb l'estructura de categories existent, i amb els usos i polítiques de Commons. Si us plau mira si hi ha un <a href="/wiki/Category:Commons_category_schemes" title="Category:Commons category schemes">esquema de categories</a> o un <a href="/wiki/Category:Commons_projects" title="Category:Commons projects">projecte de Commons</a> per al tema que tractes, i segueix les convencions allà descrites. </p>
      <p><span id="Noms_de_les_categories"></span> </p> 
      <h3><span class="mw-headline" id="Noms_de_les_categories">Noms de les categories</span></h3> 
      <p>Els noms de les categories haurien d'anar en anglès (vegeu <a href="/wiki/Commons:Language_policy/ca" title="Commons:Language policy/ca">Commons:Política lingüística</a>) Tanmateix, hi ha excepcions com ara alguns noms propis, tàxons biològics i noms per als quals el nom no anglès s'utilitza més habitualment en llengua anglesa (o no hi ha evidències de l'ús d'una versió en anglès). Els <a href="https://en.wikipedia.org/wiki/ca:Alfabets_derivats_del_llat%C3%AD" class="extiw" title="w:ca:Alfabets derivats del llatí">alfabets llatins</a> s'utilitzen en forma original, inclosos els diacrítics i les lletres derivades, els alfabets no llatins es transcriuen a l'escriptura llatina anglesa. Els caràcters bàsics en anglès (<a href="https://en.wikipedia.org/wiki/ISO/IEC_646" class="extiw" title="w:ISO/IEC 646">ISO/IEC 646</a>) es prefereixen a les variants nacionals o als conjunts de caràcters d'extensió (per exemple, apòstrofs <i>rectes</i>: ',"; envers dels <i>arrissats</i>: ‘,’,“,”), si és raonable. </p> 
      <ul>
       <li>Els tipus o grups d'objectes o persones haurien generalment d'anar en plural i en anglès sempre que sigui possible: <a href="/wiki/Category:Tools" title="Category:Tools">Category:Tools</a>, <a href="/wiki/Category:Artists" title="Category:Artists">Category:Artists</a>, <a href="/wiki/Category:Lakes" title="Category:Lakes">Category:Lakes</a>, <a href="/wiki/Category:Paintings" title="Category:Paintings">Category:Paintings</a>, <a href="/wiki/Category:Sculptures" title="Category:Sculptures">Category:Sculptures</a>, <a href="/wiki/Category:Popes" title="Category:Popes">Category:Popes</a> etc.</li> 
       <li>Els temes o activitats generals requereixen normalment un nom en la forma singular i en anglès sempre que sigui possible:: <a href="/wiki/Category:History" title="Category:History">Category:History</a>, <a href="/wiki/Category:Weather" title="Category:Weather">Category:Weather</a>, <a href="/wiki/Category:Music" title="Category:Music">Category:Music</a>, <a href="/wiki/Category:Painting" title="Category:Painting">Category:Painting</a>, <a href="/wiki/Category:Sculpture" title="Category:Sculpture">Category:Sculpture</a>, <a href="/wiki/Category:Papacy" title="Category:Papacy">Category:Papacy</a> etc.</li> 
       <li>Un objecte individual concret (una persona específica, edifici, monument, obra d'art, organització, esdeveniment, etc.) generalment utilitza una forma singular (però no sempre). <a href="https://en.wikipedia.org/wiki/ca:Nom_propi" class="extiw" title="w:ca:Nom propi">Noms propis i noms comuns</a> que no tenen una variant anglesa establerta no es tradueixen ad hoc, sinó que fan servir la forma original.</li>
      </ul> 
      <p>També hi ha una <a href="/wiki/Special:MyLanguage/Commons:Naming_categories" title="Special:MyLanguage/Commons:Naming categories">proposta fallida</a> relacionada. </p>
      <p>Les categories que agrupen les subcategories per nom generalment s'han d'anomenar "by name" en lloc de "by alphabet" (per exemple, <a href="/wiki/Category:Ships_by_name" title="Category:Ships by name">Category:Ships by name</a>). </p>
      <p>Encara no tenim una internacionalització per als noms de les categories, però aquest problema s'hauria de resoldre amb els canvis adequats al programari MediaWiki (vegeu <a href="https://phabricator.wikimedia.org/T31928" class="extiw" title="phab:T31928">T31928</a>: <i>Mostrar també els títols traduïts per idioma de l'usuari a les categories</i>). La creació d'estructures de categories entremesclades en diferents idiomes només empitjoraria les coses. </p>
      <p>Per obtenir una discussió general sobre la funció de categoria de MediaWiki, consulteu la <a href="https://meta.wikimedia.org/wiki/Help:Categories" class="extiw" title="m:Help:Categories">pàgina del manual sobre categories</a>. </p> 
      <h3><span id="Categoritzaci.C3.B3_de_les_p.C3.A0gines"></span><span class="mw-headline" id="Categorització_de_les_pàgines">Categorització de les pàgines</span></h3> 
      <p>Per afegir una categoria a una pàgina (sigui una imatge, una pàgina de galeria o una pàgina de categoria), afegiu el següent codi al final de la pàgina. </p> 
      <dl>
       <dd>
        <code>[[Category:<i>Category name</i>]]</code>
       </dd>
      </dl> 
      <p>Per exemple, si carregueu un diagrama mostrant l'òrbita dels cometes, hauríeu d'afegir el següent a la pàgina de descripció de la imatge: </p> 
      <dl>
       <dd>
        <code>[[Category:Astronomical diagrams]]<br> [[Category:Comets]]</code>
       </dd>
      </dl> 
      <p>Això farà que el diagrama aparegui a les categories <a href="/wiki/Category:Astronomical_diagrams" title="Category:Astronomical diagrams">Astronomical diagrams</a> i <a href="/wiki/Category:Comets" title="Category:Comets">Comets</a>. </p>
      <p>Per saber com trobar bones categories als fitxers que carreguis o en les subcategories en què treballis, llegeix la secció <i><a href="#Categoritzar_els_teus_fitxers">Categoritzar els teus fitxers</a></i> de més avall. </p> 
      <h3><span id="Creaci.C3.B3_d.E2.80.99una_categoria_nova"></span><span class="mw-headline" id="Creació_d’una_categoria_nova">Creació d’una categoria nova</span></h3> 
      <p>Per crear una nova categoria: </p> 
      <ol>
       <li>Feu una cerca exhaustiva per assegurar-vos que no hi ha cap categoria existent que us serveixi.</li> 
       <li>Cerqueu imatges (o una galeria o altres pàgines) que s'haurien de posar a la nova categoria. Editeu aquesta pàgina i al final inseriu la referència de la nova categoria. per exemple. <code>[[Category:Title]]</code>. Deseu la pàgina editada. La nova categoria apareix com a enllaç vermell a la part inferior de la pàgina.</li> 
       <li>Feu clic a l'enllaç vermell. La nova pàgina buida de la categoria apareix per editar-la. Ara podeu editar la categoria com qualsevol altra pàgina wiki.</li>
      </ol> 
      <p>Una pàgina de categoria ha de contenir la informació següent (per ordre d'importància): </p> 
      <ul>
       <li>Enllaços de categories que el situen en una o més categories pares. A la part inferior de la nova pàgina, inseriu línies del formulari <code>[[Category:Categories rellevants]]</code>.</li> 
       <li>Un breu text de descripció que explica què hauria d'estar a la categoria, si el títol no és prou clar o prou inequívoc. Les descripcions en idiomes particulars es poden etiquetar, per exemple. amb la plantilla <code><span style="white-space:nowrap">{{<a href="/wiki/Template:Ca" title="Template:Ca">ca</a>|...}}</span></code> per a la descripció en català, <code><span style="white-space:nowrap">{{<a href="/wiki/Template:En" title="Template:En">en</a>|...}}</span></code>per a la descripció en anglès, etc., tal com s'indica a <a href="/wiki/Special:MyLanguage/Commons:Language_templates" title="Special:MyLanguage/Commons:Language templates">Commons:Language templates</a>); o bé fent servir la plantilla <a href="/wiki/Template:Multilingual_description" title="Template:Multilingual description"><span style="font-family:monospace,monospace;">{{Multilingual description}}</span></a> per mostrar només la descripció en l'idioma preferit de l'usuari, si n'hi ha.</li> 
       <li>Els enllaços interwiki o interlingües a l'article o categoria amb el mateix tema a la Viquipèdia afegint els enllaços de lloc adequats a la pàgina de Wikidata corresponent. Després de crear la pàgina de la categoria, feu clic a "Afegeix enllaços" a la part inferior de la barra lateral a l'esquerra per afegir-los.</li> 
       <li>Si la categoria s'ha d'ordenar segons una cadena diferent del títol de la categoria, afegiu un <code>{{DEFAULTSORT:}}</code>. Per exemple, el títol d'una categoria sobre una persona no seria la cadena d'ordenació adequada. Per a aquestes categories, inseriu just abans de les categories una línia com <code>{{DEFAULTSORT:Cognom, Nom}}</code> amb la cadena d'ordenació correcta. Vegeu <a href="https://meta.wikimedia.org/wiki/Categories#Sort_key" class="extiw" title="meta:Categories">meta:Categories#Sort key</a> per obtenir més informació.</li>
      </ul> 
      <p>Per a una discussió general de la funció de categories de MediaWiki, vegeu la <a href="https://meta.wikimedia.org/wiki/Help:Categories" class="extiw" title="m:Help:Categories">pàgina del manual sobre categories</a>. </p> 
      <h3><span class="mw-headline" id="Canvi_de_nom_de_categories">Canvi de nom de categories</span></h3> 
      <p>Vegeu <a href="/wiki/Commons:Rename_a_category/ca" title="Commons:Rename a category/ca">Commons:Canvi de nom de les categories</a>. </p> 
      <h2><span id="Per_una_categoritzaci.C3.B3_m.C3.A9s_apropiada"></span><span class="mw-headline" id="Per_una_categorització_més_apropiada">Per una categorització més apropiada</span></h2> 
      <p>Les pàgines (incloses les pàgines de categories) es classifiquen segons el tema, i no el contingut, perquè els continguts no són generalment una característica permanent de la pàgina de la categoria; en particular, podeu trobar momentàniament continguts inadequats en una pàgina de categoria. </p>
      <p>Exemple: suposem que <a href="/wiki/Category:Spheres" title="Category:Spheres">Category:Spheres</a> només conté imatges de boles de cristall. No heu d'afegir <a href="/wiki/Category:Glass" title="Category:Glass">Category:Glass</a> a la pàgina de la categoria, segons el contingut actual, perquè podeu tenir esferes fetes amb una gran varietat de materials. Normalment, qualsevol imatge que mostri un objecte de vidre ja es classificaria en <a href="/wiki/Category:Glass" title="Category:Glass">Category:Glass</a> (o en una categoria de la seva subestructura). Per tant, si <a href="/wiki/Category:Spheres" title="Category:Spheres">Category:Spheres</a> està realment plena de fotografies de boles de vidre, seria una millor idea crear una pàgina de categoria nova, com ara Category:Glass spheres o Category:Crystal balls, categoritzades dins de <a href="/wiki/Category:Spheres" title="Category:Spheres">Category:Spheres</a> i <a href="/wiki/Category:Glass" title="Category:Glass">Category:Glass</a>. </p>
      <p>En general, els fitxers només haurien d'estar a la categoria més específica que existeix per a cert tema. Per exemple, els fitxers de <a href="/wiki/Category:Looking_up_the_center_of_the_Eiffel_Tower" title="Category:Looking up the center of the Eiffel Tower">Category:Looking up the center of the Eiffel Tower</a> no haurien d'estar en <a href="/wiki/Category:Paris" title="Category:Paris">Category:Paris</a> <i>(vegeu <a href="#Sobrecategorizació">sobrecategorizació</a> més avall)</i>. Si no trobeu cap categoria que s'adapti al vostre propòsit, podeu crear-la, però primer llegiu la secció sobre l'ús de categories. </p>
      <p>Això no vol dir que una imatge només ha de pertànyer a una categoria; només vol dir que les imatges no han de pertànyer a categories redundants o no específiques. Per exemple, la imatge d'un ós polar rescatat d'un iceberg per un helicòpter hauria de pertànyer a la <a href="/wiki/Category:Ursus_maritimus" title="Category:Ursus maritimus">Category:Ursus maritimus</a>, <a href="/wiki/Category:Icebergs" title="Category:Icebergs">Category:Icebergs</a> i <a href="/wiki/Category:Rescue_helicopters" title="Category:Rescue helicopters">Category:Rescue helicopters</a>. Tanmateix, no hauria de pertànyer a <a href="/wiki/Category:Ursidae" title="Category:Ursidae">Category:Ursidae</a>, <a href="/wiki/Category:Sea_ice" title="Category:Sea ice">Category:Sea ice</a> o <a href="/wiki/Category:Aircraft" title="Category:Aircraft">Category:Aircraft</a>.. </p> 
      <h3><span id="Consells_de_categoritzaci.C3.B3"></span><span class="mw-headline" id="Consells_de_categorització">Consells de categorització</span></h3> 
      <p>Les categories (o galeries) que trieu per a les càrregues han de respondre al màxim de preguntes següents: </p> 
      <ul>
       <li><b>què? / qui?</b>: què o qui mostra l'arxiu? Quin és el tema principal? Quines són les característiques més destacades de la imatge? Per exemple, <a href="/wiki/Category:Automobiles" title="Category:Automobiles">Category:Automobiles</a>, <a href="/wiki/Category:Felis_silvestris_catus" title="Category:Felis silvestris catus">Category:Felis silvestris catus</a> o <a href="/wiki/Category:Jimmy_Wales" title="Category:Jimmy Wales">Category:Jimmy Wales</a></li> 
       <li><b>on?</b>: on es va fer la imatge? Quina és la ubicació del tema? Quina és la ubicació de la càmera? Això és especialment important per a imatges de llocs. Per exemple. <a href="/wiki/Category:Basin_Street,_New_Orleans" title="Category:Basin Street, New Orleans">Category:Basin Street, New Orleans</a> 
        <dl>
         <dd>
          també utilitzeu <a href="/wiki/Template:Location" title="Template:Location"><span style="font-family:monospace,monospace;">{{Location}}</span></a>
         </dd>
        </dl></li> 
       <li><b>quan?</b>: quan van ocórrer els fets representats o quan es va crear la imatge? Quan es va fer la imatge? Això és especialment important per a les imatges històriques. Un exemple seria Categoria: <a href="/wiki/Category:Warsaw_in_September_1939" title="Category:Warsaw in September 1939">Category:Warsaw in September 1939</a>, <a href="/wiki/Category:April_2010_in_Northern_Ireland" title="Category:April 2010 in Northern Ireland">Category:April 2010 in Northern Ireland</a></li> 
       <li><b>qui?</b>: qui és l'autor? Això és especialment important per a obres d'artistes coneguts i per a imatges històriques, per exemple <a href="/wiki/Category:Paintings_by_Rembrandt" title="Category:Paintings by Rembrandt">Category:Paintings by Rembrandt</a>. També podeu utilitzar les pàgines de l'<a href="/wiki/Category:Creator_templates" title="Category:Creator templates">espai de noms <i>Creator</i></a> com a plantilles per aconseguir-ho.</li> 
       <li><b>com?</b>: com ho fa el fitxer (o la imatge)? Concretament: 
        <ul>
         <li><b>quina vista?</b>: quin tipus de visualització mostra la imatge? per exemple. <a href="/wiki/Category:Plan_views" title="Category:Plan views">Category:Plan views</a>, <a href="/wiki/Category:Panoramics" title="Category:Panoramics">Category:Panoramics</a>.</li> 
         <li><b>quin color?</b>: quin és el color general? per exemple: <a href="/wiki/Category:Yellow_animals" title="Category:Yellow animals">Category:Yellow animals</a></li> 
         <li><b>quina tècnica de fotografia?</b>: Si la imatge utilitza una tècnica o efecte específic, apliqueu la categoria corresponent: p. ex. <a href="/wiki/Category:Tone-mapped_HDR_images" title="Category:Tone-mapped HDR images">Category:Tone-mapped HDR images</a>, <a href="/wiki/Category:Black_and_white_photographs" title="Category:Black and white photographs">Category:Black and white photographs</a>, <a href="/wiki/Category:Double_exposure" title="Category:Double exposure">Category:Double exposure</a></li>
        </ul></li>
      </ul> 
      <p>Les preguntes anteriors cobreixen els principals aspectes de la imatge a classificar. Per a algunes imatges té sentit utilitzar-les totes, per a altres imatges només una o dues són raonables. A més, hi ha altres aspectes de les imatges que es poden utilitzar per classificar la imatge: </p> 
      <ul>
       <li><b>quina font?:</b> informació sobre la procedència de la imatge. Per exemple, <a href="/wiki/Category:Images_from_the_German_Federal_Archive" title="Category:Images from the German Federal Archive">Category:Images from the German Federal Archive</a></li> 
       <li><b>quin format?:</b> informació sobre el tipus de suport si és inusual. Per exemple, <a href="/wiki/Category:Audio" title="Category:Audio">Category:Audio</a> or <a href="/wiki/Category:Animated_GIF" title="Category:Animated GIF">Category:Animated GIF</a>, <a href="/wiki/Category:SVG" class="mw-disambig" title="Category:SVG">Category:SVG</a></li> 
       <li><b>quin programari?:</b> informació sobre el programari utilitzat per crear la imatge. Per exemple, <a href="/wiki/Category:Created_with_Hugin" title="Category:Created with Hugin">Category:Created with Hugin</a></li> 
       <li><b>quina càmera?:</b> informació sobre la càmera. Per exemple, <a href="/wiki/Category:Taken_with_Nikon_D80" title="Category:Taken with Nikon D80">Category:Taken with Nikon D80</a></li>
      </ul> 
      <p>Aquest darrer conjunt és útil i important, però sempre s'ha de fer a més del conjunt principal de criteris. </p>
      <p>La categorització a Wikimedia Commons és més detallada i profunda que la classificació en els projectes de Viquipèdia. En comparació amb ells, Commons té més categories per a temes individuals: llocs, persones, organitzacions, esdeveniments, termes, etc. Gairebé tots els articles d'una Viquipèdia poden tenir una categoria corresponent a Commons. Tanmateix, fins i tot si hi ha més imatges d'una persona ordinària o d'un esdeveniment incidental, és pràctic agrupar-les en una categoria especial i classificar aquesta categoria en lloc de classificar totes les imatges similars individualment en un conjunt idèntic de categories parentals. </p> 
      <h3><span class="mw-headline" id="Trobar_una_categoria_apropiada">Trobar una categoria apropiada</span></h3> 
      <p>Per trobar les categories adequades per a les vostres càrregues, heu de navegar per l'estructura de categories a partir d'una categoria genèrica. Limiteu la cerca a subcategories fins que trobeu la categoria més específica que s'adapti al fitxer que heu penjat. Podeu navegar per l'estructura de les categories seguint enllaços a subcategories o ampliant l'arbre de subcategories fent clic als símbols + dels petits noms de subcategories. La secció anterior <i><a href="#Categories_principals">Categories principals</a></i> proporciona un punt de partida i la <i><a href="#Com_categoritzar:_orientació_per_temes">Com categoritzar: orientació per temes</a></i> cobreix alguns temes més. </p> 
      <h3><span id="Sobrecategorizaci.C3.B3"></span><span class="mw-headline" id="Sobrecategorizació">Sobrecategorizació</span></h3>

      <p>La sobrecategorizació és el que passa quan una imatge apareix en diverses categories en la mateixa línia d'un arbre. La regla general <i>se situa sempre una imatge en les categories més específiques i no en els nivells superiors</i>. Les excepcions a aquesta regla s’expliquen a <a href="#OvercatException">la secció següent</a>. </p>
      <p>Un exemple: Una imatge que s'ha de classificar mostra un cercle groc. Aquesta imatge s'hauria de col·locar dins <a href="/wiki/Category:Yellow_circles" title="Category:Yellow circles">Category:Yellow circles</a>. Si també es col·loca a <a href="/wiki/Category:Circles" title="Category:Circles">Category:Circles</a>, es categoritza en excés. Ja sabem que és un cercle, perquè tots els cercles grocs són cercles. Per tant, <a href="/wiki/Category:Circles" title="Category:Circles">Category:Circles</a> és redundant. Es pot utilitzar <a href="/wiki/Template:Uw-overcat" title="Template:Uw-overcat">Template:Uw-overcat</a> per aconsellar-ho als usuaris. </p>
      <p>Això s'aplica a la majoria de fitxers: Com s'explica més amunt, els fitxer de <a href="/wiki/Category:Black_and_white_photographs_of_the_Eiffel_Tower" title="Category:Black and white photographs of the Eiffel Tower">Category:Black and white photographs of the Eiffel Tower</a> no haurien d'estar a <a href="/wiki/Category:Paris" title="Category:Paris">Category:Paris</a>, els fitxers de <a href="/wiki/Category:Albert_Einstein" title="Category:Albert Einstein">Category:Albert Einstein</a> no haurien d'estar a <a href="/wiki/Category:Physicists_from_Germany" title="Category:Physicists from Germany">Category:Physicists from Germany</a> i així successivament. </p> 
      <h4><span id="Per_qu.C3.A8_la_sobrecategorizaci.C3.B3_constitueix_un_problema"></span><span class="mw-headline" id="Per_què_la_sobrecategorizació_constitueix_un_problema">Per què la sobrecategorizació constitueix un problema</span></h4> 
      <p>Sovint es pensa que com més categories tingui una imatge, més fàcil serà trobar-la. Segons aquesta lògica, tota imatge d'un home hauria d'estar a <a href="/wiki/Category:Men" title="Category:Men">Category:Men</a>, perquè encara que un usuari sàpiga una mica més de la persona que està buscant, a part que és un home, seria capaç de trobar-lo en la categoria general. El resultat és que aquesta categoria superior s'omple, fent necessari passar per centenars o milers d'imatges, fins a trobar la que es vol. Probablement, l'usuari no trobarà la que busca, i el que és més, els que busquen una fotografia genèrica d'un home per il·lustrar l'article <a href="https://ca.wikipedia.org/wiki/Home" class="extiw" title="ca:Home">ca:Home</a> trobarà que estan sepultades entre estrelles de cinema, científics i polítics. </p>
      <p>En nivells més baixos, el problema és menys greu, ja que la quantitat d'imatges serà menor - encara que poden fàcilment arribar a centenars. Però segueix havent-hi un problema: Tornant a l'exemple, si un usuari busca una imatge d'Einstein i sap que és un físic, buscarà a <a href="/wiki/Category:Physicists" title="Category:Physicists">Category:Physicists</a>. Si troba una imatge entre els centenars d'aquesta categoria, encara que no sigui la que més el satisfà, pot quedar-se amb ella, en lloc de seguir buscant per la subcategoria <a href="/wiki/Category:Physicists_from_Germany" title="Category:Physicists from Germany">Category:Physicists from Germany</a> fins a arribar a <a href="/wiki/Category:Albert_Einstein" title="Category:Albert Einstein">Category:Albert Einstein</a>, on pot haver una altra millor. Així, la sobrecategorizació ha causat dos problemes, la categoria superior està saturada, i els usuaris poden deixar de buscar la categoria més rellevant, ja que han trobat una que té una imatge més o menys interessant. </p> 
      <h4><span id="Categoritzaci.C3.B3_incorrecta_de_categories"></span><span class="mw-headline" id="Categorització_incorrecta_de_categories">Categorització incorrecta de categories</span></h4> 
      <p>Encara que, sembli estrany, la subcategorització és una causa de sobrecategorizació. Això passa quan una categoria no categoritza adequadament, fent que els usuaris sobrecategoritzin una imatge perquè encaixi en les categories més rellevants. Per exemple, <a href="/wiki/Category:Eiv%C3%B8r_P%C3%A1lsd%C3%B3ttir" title="Category:Eivør Pálsdóttir">Category:Eivør Pálsdóttir</a> es va incloure només en <a href="/wiki/Category:People_by_name" title="Category:People by name">Category:People by name</a>. Així que si algú afegeix imatges d'ella, i sap qui és, col·locarà les imatges a <a href="/wiki/Category:Female_vocalists_from_the_Faroe_Islands" title="Category:Female vocalists from the Faroe Islands">Category:Female vocalists from the Faroe Islands</a>. En aquest moment, però, totes les imatges que ja estiguessin col·locades en les dues categories estarien sobrecategoritzades i haurien de ser eliminades manualment de la categoria pare. </p>
      <p>Un problema relacionat és la categorització errònia. Notting Hill és un districte del barri (borough) de Kensington i Chelsea de Londres. Quan es va crear <a href="/wiki/Category:Notting_Hill" title="Category:Notting Hill">Category: Notting Hill</a> va estar més d'un mes ubicada a <a href="/wiki/Category:London" title="Category:London">Category:London</a>. Això incitava a que en afegir una imatge, es col·loqués directament a <a href="/wiki/Category:Royal_Borough_of_Kensington_and_Chelsea" title="Category:Royal Borough of Kensington and Chelsea">Category:Royal Borough of Kensington and Chelsea</a>, que és el districte del Gran Londres en què es troba Notting Hill, en no trobar la categoria de Notting Hill dins de Kensington and Chelsea quan se seguia l'arbre dels districtes de Londres. En lloc d'això, cada imatge només hauria d'estar en les categories més específiques, i aquestes categories haurien al seu torn col·locar les categories més específiques que les poguessin albergar. </p>
      <p>Si trobeu categories incorrectament categoritzades, si us plau, col·loqueu-les a les categories parentals adequades. Això no només ajudarà a evitar una categorització excessiva, sinó que també facilitarà el desplaçament per l'arbre de categories. </p> 
      <h4><span id="Excepci.C3.B3_per_a_imatges_amb_m.C3.A9s_temes_categoritzats"></span><span class="mw-headline" id="Excepció_per_a_imatges_amb_més_temes_categoritzats"><span id="OvercatException"></span> Excepció per a imatges amb més temes categoritzats</span></h4> 
      <p>Un fitxer que només representi un tema rellevant no s'ha de sobrecategoritzar. Quan un fitxer mostra temes rellevants addicionals i els temes addicionals no tenen les seves pròpies subcategories, es pot considerar la possibilitat de categoritzar temporalment la imatge tant en la subcategoria com en la categoria pare. </p>
      <p>Per exemple, aquesta situació pot sorgir en el cas d'una fotografia de tres polítics, un dels quals és <a href="/wiki/Category:Angela_Merkel" title="Category:Angela Merkel">Angela Merkel</a> (que té la seva pròpia categoria Commons), amb altres dos polítics que encara no tenen les seves pròpies categories. Tot i que, sens dubte, la imatge es categoritzaria en <a href="/wiki/Category:Angela_Merkel" title="Category:Angela Merkel">Category:Angela Merkel</a> o en una de les seves subcategories, normalment es consideraria una categorització excessiva incloure-la també a <a href="/wiki/Category:Politicians_of_Germany" title="Category:Politicians of Germany">Category:Politicians of Germany</a>. Tanmateix, és poc probable que els usuaris cerquin els altres dos polítics de la categoria Merkel. Idealment, crearíem subcategories específiques per als altres dos polítics (si es justifica) o trobaríem altres subcategories rellevants (per exemple, <a href="/wiki/Category:Politicians_of_Bavaria" title="Category:Politicians of Bavaria">Category:Politicians of Bavaria</a> o <a href="/wiki/Category:Members_of_the_FDP" title="Category:Members of the FDP">Category:Members of the FDP</a>, etc.), que ens permetessin evitar una sobrecategorització. En algunes circumstàncies, però, és possible que haguem de classificar temporalment la imatge a <a href="/wiki/Category:Politicians_of_Germany" title="Category:Politicians of Germany">Category:Politicians of Germany</a>, on encara no existeixen les altres subcategories adequades. </p>
      <p>Els països es poden classificar com a part de diverses categories superposades. Per exemple, <a href="/wiki/Category:India" title="Category:India">Category:India</a> es troba a <a href="/wiki/Category:Countries_of_South_Asia" title="Category:Countries of South Asia">Category:Countries of South Asia</a>, així com a <a href="/wiki/Category:Countries_of_Asia" title="Category:Countries of Asia">Category:Countries of Asia</a>. </p>
      <p>També les categories d'usuaris queden exemptes de la sobrecategorització, ja que no són visibles per a la majoria dels espectadors i els usuaris del projecte les inclouen per a molts propòsits diferents, com ara ordenar, estadístiques, omplir valors per a les caixes d'usuaris, etc. </p> 
      <h2><span id="Com_categoritzar:_orientaci.C3.B3_per_temes"></span><span class="mw-headline" id="Com_categoritzar:_orientació_per_temes">Com categoritzar: orientació per temes</span></h2> 
      <p>Per a algunes categories, hi ha indicacions especials sobre la millor manera d'ordenar el contingut dins d'aquesta categoria. Aquesta guia es pot trobar en un <a href="/wiki/Category:Commons_category_schemes" title="Category:Commons category schemes">esquema de categories</a> o en un <a href="/wiki/Category:Commons_projects" title="Category:Commons projects">projecte de Commons</a> per al vostre tema. També hi ha informació de classificació en aquesta secció i, de vegades, hi ha indicacions a la part superior de la pàgina de la categoria, en la a Categoria <a href="https://meta.wikimedia.org/wiki/Namespace" class="extiw" title="meta:Namespace">espai de noms</a>. Per exemple, algunes instruccions sobre la classificació del contingut que representa persones es troben a la part superior de <a href="/wiki/Category:People" title="Category:People">Category:People</a>, i algunes en la secció <a href="#Persones">Persones</a> que hi ha a continuació. </p> 
      <h3><span class="mw-headline" id="Persones">Persones</span></h3> 
      <p>El contingut que representa persones s'ha de classificar en categories que les descriuen, com ara <a href="/wiki/Category:Economists_from_the_United_States" title="Category:Economists from the United States">Category:Economists from the United States</a>. Comenceu a explorar <a href="/wiki/Category:People" title="Category:People">Category:People</a>. </p>
      <p>Consulteu <a href="/wiki/Special:MyLanguage/Commons:Category_scheme_People" title="Special:MyLanguage/Commons:Category scheme People">Commons:Esquema de la categoria Persones</a> per obtenir detalls sobre com nomenar i organitzar aquestes categories. </p> 
      <h3><span class="mw-headline" id="Paisatges_i_vistes_exteriors">Paisatges i vistes exteriors</span></h3> 
      <p>Els continguts que representen un tema determinat des d'un punt de vista comú s'agrupen en categories <i>Views of Subject from Viewpoint</i>, com ara <a href="/wiki/Category:Views_of_Cathedral_of_Seville_from_the_Giralda" title="Category:Views of Cathedral of Seville from the Giralda">Views of Cathedral of Seville from the Giralda</a>. Aquestes categories haurien de ser subcategories tant de la categoria del subjecte (<a href="/wiki/Category:Cathedral_of_Seville" title="Category:Cathedral of Seville">Cathedral of Seville</a> en aquest exemple) com de la categoria del mirador (<a href="/wiki/Category:Giralda" title="Category:Giralda">Giralda</a> en aquest exemple). </p>
      <p>En aquest exemple, les <a href="/wiki/Category:Views_of_Cathedral_of_Seville_from_the_Giralda" title="Category:Views of Cathedral of Seville from the Giralda">Views of Cathedral of Seville from the Giralda</a> no es col·loquen directament a les categories de subjectes i miradors, sinó a <a href="/wiki/Category:Views_of_the_Cathedral_of_Seville" title="Category:Views of the Cathedral of Seville">Views of the Cathedral of Seville</a> i <a href="/wiki/Category:Views_from_Giralda" title="Category:Views from Giralda">Views from Giralda</a>. Aquestes categories intermèdies sovint són necessàries per crear estructura i evitar una <a href="#Sobrecategorizació">sobrecategorizació</a>, particularment per a les vistes d'una ciutat des d'un mirador situat dins de la ciutat. Per exemple, <a href="/wiki/Category:Views_of_Rome_from_the_Pincio" title="Category:Views of Rome from the Pincio">Views of Rome from the Pincio</a> necessita la categoria intermèdia <a href="/wiki/Category:Views_of_Rome" title="Category:Views of Rome">Views of Rome</a> per evitar situar-la directament a <a href="/wiki/Category:Rome" title="Category:Rome">Rome</a>, cosa que constituiria una sobrecategorizació. </p> 
      <h3><span class="mw-headline" id="Textos">Textos</span></h3> 
      <p>Els textos, com ara les exploracions de llibres, normalment haurien de tenir una categoria per a cada versió de l'exploració i per a cada edició del text. Per tant, un llibre publicat en tres edicions separades tindria una categoria principal per al llibre, tres subcategories per a cada text i altres subcategories per al text com a jpeg, DjVu, etc., suposant que cada versió s'hagués carregat. (Les categories no es crearien per a edicions que no es mantinguin a Commons.) Això és particularment important per a fitxers en formats diferents de DjVu i PDF, on la categoria és l'únic mitjà pràctic per mantenir els escaneigs units; veure per exemple <a href="/wiki/Category:The_Chronicles_of_England,_Scotland_and_Ireland,_Holinshed,_1587" title="Category:The Chronicles of England, Scotland and Ireland, Holinshed, 1587">Category:The Chronicles of England, Scotland and Ireland, Holinshed, 1587</a> que conté 2857 imatges jpeg d'exploracions de pàgines. </p>

       <h3><span class="mw-headline" id="GLAMs">GLAMs</span></h3> 
      </div> 
      <p>Per a qüestions de categorització relacionades amb les donacions de contingut massiu de GLAM (galeries, biblioteques, arxius i museus), consulteu <a href="/wiki/Special:MyLanguage/Commons:Guide_to_batch_uploading#Categories" title="Special:MyLanguage/Commons:Guide to batch uploading">Commons:Guide to batch uploading#Categories</a>. </p> 
      <h2><span id="Flux_de_treball_de_categoritzaci.C3.B3"></span><span class="mw-headline" id="Flux_de_treball_de_categorització">Flux de treball de categorització</span></h2> 
      <p>Actualment, un bot verifica si els fitxers penjats recentment es classifiquen en categories temàtiques i intenta classificar els fitxers que no hi són. Abans del 17 de juny de 2015, <a href="/wiki/User:CategorizationBot" title="User:CategorizationBot">CategorizationBot</a> era el responsable d'aquesta feina. A partir de juny de 2019, <a href="/wiki/User:SteinsplitterBot" title="User:SteinsplitterBot">SteinsplitterBot</a> ocasionalment comprova si hi ha fitxers no classificats. El flux de treball és el següent: </p> 
      <ol>
       <li>L'usuari penja un fitxer nou i afegeix categories (o no).</li> 
       <li>Un bot verifica si el fitxer està classificat. 
        <ul>
         <li>El fitxer ja està categoritzat =&gt; d'acord</li> 
         <li>El fitxer no està categoritzat =&gt; el bot etiqueta el fitxer a <a href="/wiki/Category:Media_needing_categories" title="Category:Media needing categories">Category:Media needing categories</a>. Els fitxers d'avui es troben a <a href="/wiki/Category:Media_needing_categories_as_of_6_June_2021" title="Category:Media needing categories as of 6 June 2021">Category:Media needing categories as of 6 June 2021</a></li>
        </ul></li> 
       <li>Els usuaris classifiquen més tard els fitxers (p. Ex., Difusió de la categoria a continuació)</li>
      </ol> 
      <p><i>Vegeu també:</i> <a href="/wiki/User:CategorizationBot#Process" title="User:CategorizationBot">User:CategorizationBot#Process</a>, <a href="/wiki/User:Multichill/Categorization_stats" title="User:Multichill/Categorization stats">categorization statistics</a> </p>
      <p>Es poden fer altres fluxos de treball de classificació manuals: </p> 
      <ul>
       <li>Emplenament de categories: Utilitzeu paraules clau adequades al motor de cerca per trobar els fitxers que haurien d'estar en una categoria determinada i col·locar-los allà.</li> 
       <li>Difusió de categories: Aneu a <a href="/wiki/Category:Categories_requiring_diffusion" title="Category:Categories requiring diffusion">Category:Categories requiring diffusion</a>, seleccioneu una categoria adient, creeu subcategories adequades si cal i moveu els fitxers a les subcategories. Els ginys com Cat-a-lot i Hotcats us poden ajudar.</li>
      </ul> 
      <h2><span id="Categories_amb_la_marca_.C2.ABHIDDENCAT.C2.BB"></span><span class="mw-headline" id="Categories_amb_la_marca_«HIDDENCAT»">Categories amb la marca «HIDDENCAT»</span></h2> 
      <div class="shortcut2" style="">
       <link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r433650343"><a href="/wiki/Commons:Shortcuts" title="Commons:Shortcuts">Shortcut</a>
       <br><a href="/wiki/Commons:HIDDENCAT" class="mw-redirect" title="Commons:HIDDENCAT">COM:HIDDENCAT</a>
      </div> 
      <p>Moltes categories no temàtiques estan marcades amb <code>__HIDDENCAT__</code> o {{<a href="/wiki/Template:Hiddencat" title="Template:Hiddencat">Hiddencat</a>}} a la pàgina de la categoria. Per exemple, vegeu <a class="external text" href="https://commons.wikimedia.org/w/index.php?title=Category:PD_NASA&amp;action=edit">Category:PD NASA in edit mode</a>. </p>
      <p>Tot i que les categories són visibles generalment a totes les pàgines, les categories marcades amb <code>__HIDDENCAT__</code> només són visibles: </p> 
      <ul>
       <li>en la pantalla d'edició: al final de la pantalla, a sota del quadre d'edició</li> 
       <li>en les pàgines de categories: 
        <ul>
         <li>en subcategories a la categoria oculta: a la ubicació normal, però en una línia separada amb una tipografia més petita i l'etiqueta "Hidden categories."</li> 
         <li>en les categories pares: de la mateixa manera que altres categories</li>
        </ul></li> 
       <li>en les pàgines de descripció del fitxer i a les pàgines de la galeria: per als usuaris connectats que han seleccionat "Mostrar categories ocultes" a les seves <a href="/wiki/Special:Preferences#mw-prefsection-rendering" title="Special:Preferences">preferències d'aparença</a>. S'activa per a tots els usuaris registrats recentment.</li>
      </ul> 
      <p>Aquesta característica s'utilitza generalment per a categories basades en plantilles, com ara categories basades en etiquetes de llicència. Per exemple, si col·loqueu <a href="/wiki/Template:PD-old-100" title="Template:PD-old-100"><span style="font-family:monospace,monospace;">{{PD-old-100}}</span></a> a una pàgina de descripció del fitxer, s'afegeix el fitxer a <a href="/wiki/Category:Author_died_more_than_100_years_ago_public_domain_images" title="Category:Author died more than 100 years ago public domain images">Category:Author died more than 100 years ago public domain images</a>, que està marcat amb <code>__HIDDENCAT__</code>. </p>
      <p>Per obtenir més informació, consulteu la <a href="https://meta.wikimedia.org/wiki/Help:Category#Hidden_categories" class="extiw" title="m:Help:Category">secció d’ajuda sobre categories ocultes</a> de Mediawiki (el programari que utilitza Commons). </p> 
      <h2><span class="mw-headline" id="Plantilles_per_a_les_categories">Plantilles per a les categories</span></h2> 
      <div lang="en" dir="ltr" class="mw-content-ltr"> 
       <p>Some templates are designed for use on category pages - see <a href="/wiki/Category:Category_namespace_templates" title="Category:Category namespace templates">Category:Category namespace templates</a>. If the category is linked to a Wikidata entry, then you can use: </p> 
       <ul>
        <li><a href="/wiki/Template:Wikidata_Infobox" title="Template:Wikidata Infobox"><span style="font-family:monospace,monospace;">{{Wikidata Infobox}}</span></a></li>
       </ul> 
       <p>which displays a summary of the topic's information that is available on Wikidata, and also auto-adds birth/death/name/monument ID categories. </p> 
      </div> 
      <div lang="en" dir="ltr" class="mw-content-ltr"> 
       <p>Some of the more commonly used ones are <a href="/wiki/Category:Category_header_templates" title="Category:Category header templates">Category:Category header templates</a> such as: </p> 
       <ul>
        <li><a href="/wiki/Template:Category_TOC" class="mw-redirect" title="Template:Category TOC"><span style="font-family:monospace,monospace;">{{Category TOC}}</span></a> and <a href="/wiki/Template:LargeCategoryTOC" title="Template:LargeCategoryTOC"><span style="font-family:monospace,monospace;">{{LargeCategoryTOC}}</span></a> - to provide a table of contents for very large categories with many subcategories</li> 
        <li><a href="/wiki/Template:ImageTOC" title="Template:ImageTOC"><span style="font-family:monospace,monospace;">{{ImageTOC}}</span></a> and <a href="/wiki/Template:FileCategoryTOC" title="Template:FileCategoryTOC"><span style="font-family:monospace,monospace;">{{FileCategoryTOC}}</span></a> - to provide a table of contents for large categories with many files and few subcategories</li> 
        <li><a href="/wiki/Template:Image_template_notice" title="Template:Image template notice"><span style="font-family:monospace,monospace;">{{Image template notice}}</span></a> and <a href="/wiki/Template:Autocat" title="Template:Autocat"><span style="font-family:monospace,monospace;">{{Autocat}}</span></a> - for maintenance categories populated by a specific template.</li> 
        <li><a href="/wiki/Template:Global_maintenance_category" title="Template:Global maintenance category"><span style="font-family:monospace,monospace;">{{Global maintenance category}}</span></a> and <a href="/wiki/Template:Local_maintenance_category" title="Template:Local maintenance category"><span style="font-family:monospace,monospace;">{{Local maintenance category}}</span></a></li>
       </ul> 
      </div> 
      <div lang="en" dir="ltr" class="mw-content-ltr"> 
       <ul>
        <li><a href="/wiki/Template:MetaCat" title="Template:MetaCat"><span style="font-family:monospace,monospace;">{{MetaCat}}</span></a> - for categories which should only contain other categories 
         <ul>
          <li><a href="/wiki/Template:By_country_category" title="Template:By country category"><span style="font-family:monospace,monospace;">{{By country category}}</span></a>, <a href="/wiki/Template:By_language_category" title="Template:By language category"><span style="font-family:monospace,monospace;">{{By language category}}</span></a>, <a href="/wiki/Template:By_century_category" title="Template:By century category"><span style="font-family:monospace,monospace;">{{By century category}}</span></a> - for specific types of meta category</li>
         </ul></li> 
        <li><a href="/wiki/Template:CatDiffuse" title="Template:CatDiffuse"><span style="font-family:monospace,monospace;">{{CatDiffuse}}</span></a> (temporary) and <a href="/wiki/Template:Categorise" title="Template:Categorise"><span style="font-family:monospace,monospace;">{{Categorise}}</span></a> (permanent) - for large categories which need maintenance to move files into subcategories</li> 
        <li><a href="/wiki/Template:Category_tree" title="Template:Category tree"><span style="font-family:monospace,monospace;">{{Category tree}}</span></a></li>
       </ul> 
      </div>

        <li>Gadgets enabled through the <a href="/wiki/Special:Preferences#mw-prefsection-gadgets" title="Special:Preferences">user preferences</a> 
         <ul>
          <li><i>Cat-a-lot</i>: A tool that helps with moving multiple files between categories or adding categories to search results. <small>[<a href="/wiki/Help:Gadget-Cat-a-lot" title="Help:Gadget-Cat-a-lot">documentation</a>&nbsp;/&nbsp;<a href="/wiki/MediaWiki_talk:Gadget-Cat-a-lot.js" title="MediaWiki talk:Gadget-Cat-a-lot.js">talk</a>]</small>&nbsp;</li>

          <li><i>Gallery Details</i>: Adds a link in the toolbox to display galleries and categories (and <a href="/wiki/Special:NewFiles" title="Special:NewFiles">Newimages</a> and <a href="/wiki/Special:Search" title="Special:Search">Search</a> result pages) with extensive details from file description pages and links to easily mark an image without source, etc. If <i>Pretty log</i> is activated, it also works on <a href="/wiki/Special:Log" title="Special:Log">Log</a> pages. <small>[<a href="/wiki/Help:Gadget-GalleryDetails" title="Help:Gadget-GalleryDetails">documentation</a>&nbsp;/&nbsp;<a href="/wiki/MediaWiki_talk:Gadget-GalleryDetails.js" title="MediaWiki talk:Gadget-GalleryDetails.js">talk</a>]</small>&nbsp;</li> 
          <li>Place categories above content, but below image on file description pages. &nbsp;Modifies the placement of categories on the user interface.</li> 
          <li>Add a link to category pages to search for the category name with the option "-incategory". This excludes files already in the category (doesn't work if the category was added by a template). <small>[<a href="/wiki/MediaWiki_talk:Gadget-Searchnotincat.js" title="MediaWiki talk:Gadget-Searchnotincat.js">talk</a>]</small>&nbsp;</li>
