      <h2><span id="What_is_the_Tango_Desktop_Project?"></span><span class="mw-headline" id="What_is_the_Tango_Desktop_Project.3F">What is the Tango Desktop Project?</span></h2> 
      <p><b>The Tango Desktop Project</b> exists to help create a consistent graphical user interface experience for free and Open Source software. </p>
      <p><br> </p>
      <div style="width: 480px; margin: auto;"> 
       <img src="/static/links.png" width="459" height="159" border="0" usemap="#map"> <map name="map"> <area shape="rect" coords="0,1,134,153" href="/Tango_Icon_Theme_Guidelines"> <area shape="rect" coords="160,0,304,154" href="/Tango_Icon_Library"> <area shape="rect" coords="311,1,458,158" href="/Tango_Showroom"> </map> 
      </div> 
      <p class="mw-empty-elt"></p>
      <p>While the look and feel of an application is determined by many individual components, some organization is necessary in order to unify the appearance and structure of individual icon sets used within those components. </p>
      <p>The Tango Desktop Project defines an <a href="/Tango_Icon_Theme_Guidelines" title="Tango Icon Theme Guidelines">icon style guideline</a> to which artists and designers can adhere. A sample implementation of the style is available as an <a href="/Tango_Icon_Library" title="Tango Icon Library"> icon theme</a> based upon a standardized <a href="/Standard_Icon_Naming_Specification" title="Standard Icon Naming Specification"> icon naming specification</a>. In addition, the project provides transitional utilities to assist in creating icon themes for existing desktop environments, such as <a rel="nofollow" class="external text" href="http://www.gnome.org">GNOME</a> and <a rel="nofollow" class="external text" href="http://www.kde.org">KDE</a>. </p>
      <p><br> Eventually, the Tango Desktop Project will provide the following initiatives: </p> 
      <ul>
       <li>A suggested default native look.</li> 
       <li>A subsystem to help standardize toolkits on a common icon naming structure.</li> 
       <li>A complete set of application, mimetype, and stock <a href="/Tango_Icon_Library" title="Tango Icon Library"> icons</a> designed using a defined <a href="/Tango_Icon_Theme_Guidelines" title="Tango Icon Theme Guidelines"> style guide</a>.</li> 
       <li>A general, cross-desktop human interface guideline.</li>
      </ul> 
      <h2><span id="Who_is_involved?"></span><span class="mw-headline" id="Who_is_involved.3F">Who is involved?</span></h2> 
      <p>We are <a href="/The_People" title="The People">engineers, artists, user interface designers, and volunteers</a>. We'd like you to be involved. </p>
      <p>Join us in our effort to produce a native, open look and structure by contributing to the guidelines, improving applications (by encouraging them follow the proposed specifications), contributing artwork, and by spreading the ideas expressed here. </p>
      <p>Interested contributors from <a rel="nofollow" class="external text" href="http://www.gnome.org/">Gnome</a>, <a rel="nofollow" class="external text" href="http://www.kde.org/">KDE</a>, and other <a rel="nofollow" class="external text" href="http://www.fsf.org/">free</a> and <a rel="nofollow" class="external text" href="http://www.opensource.org/">open source</a> projects are welcome. We invite anyone to follow the specifications defined here, as well as those defined on <a rel="nofollow" class="external text" href="http://freedesktop.org/">freedesktop.org</a>. </p>
      <p>You can file bugs against the Tango icon library, icon style guideline, and the Icon Naming Specification in <a rel="nofollow" class="external text" href="http://bugzilla.freedesktop.org/">freedesktop.org's bugzilla</a>. </p>
      <p>The Tango icon library, icon style guideline matters, and artistic workflows can be discussed on the <a rel="nofollow" class="external text" href="http://lists.freedesktop.org/mailman/listinfo/tango-artists">tango-artists mailing list</a>. The Icon Naming Specification can be discussed on the <a rel="nofollow" class="external text" href="http://lists.freedesktop.org/mailman/listinfo/xdg">XDG mailing list</a>. </p>
