         <h1 id="omnibus-gitlab-docs">Omnibus GitLab Docs<a href="#omnibus-gitlab-docs" title="Permalink" class="anchor"></a> </h1> 
         <p>Omnibus GitLab is a way to package different services and tools required to run GitLab, so that most users can install it without laborious configuration. </p>
