      <p>A Wikipedia <b>log</b> is an automatically generated list of actions of a certain type, showing when they were performed, by what user, and with respect to what page or other user. Logged actions include page deletions and restorations, page moves (renames), user blocks and unblocks, page protections and unprotections, user account creations, abuse filter reports, and others. </p>
      <p>Logs can list actions done <b>by</b> a user (or IP editor), actions done <b>to</b> a user (or IP editor), and actions done <b>to</b> a wiki page. </p>

      <h2><span class="mw-headline" id="Viewing_logs">Viewing logs</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Help:Log&amp;action=edit&amp;section=1" title="Edit section: Viewing logs">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>To view Wikipedia's logs, go to <b><a href="/wiki/Special:Log" title="Special:Log">Special:Log</a></b>. From there you can decide which actions you wish to see logged, and filter by certain other parameters, such as date and time, user and affected page. When a log type is selected and you click "Show", the top of the page describes the log. You can also go directly to a selected log with a search term or link like <a href="/wiki/Special:Log/block" title="Special:Log/block">Special:Log/block</a>, which displays the block log. Other parameters that can be used in a similar way include <a href="/wiki/Special:Log/delete" title="Special:Log/delete">delete</a>, <a href="/wiki/Special:Log/move" title="Special:Log/move">move</a>, <a href="/wiki/Special:Log/protect" title="Special:Log/protect">protect</a>, <a href="/wiki/Special:Log/rights" title="Special:Log/rights">rights</a>, <a href="/wiki/Special:Log/upload" title="Special:Log/upload">upload</a> and <a href="/wiki/Special:Log/patrol" title="Special:Log/patrol">patrol</a>. </p>
      <p>A link to the log of actions relating to a particular page can be found on the "View logs for this page" link on the "<a href="/wiki/Help:Page_history" title="Help:Page history">View history</a>" tab at top of the page. A link to the log of actions by a particular user can be found on the left-hand side when the user's talk page or user page is opened. Remember, however, that these are actions <i>performed</i> by that user; for actions affecting a given user (such as blocks), the username (preceded by "User:") must be entered in the "Title" field of the log page, not the "User" field. (This configuration can be accessed directly from the "View history" tab of the user's userpage.) </p> 
      <h2><span class="mw-headline" id="Using_a_log">Using a log</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Help:Log&amp;action=edit&amp;section=2" title="Edit section: Using a log">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p><a href="/wiki/File:Screenshot_delete_log_of_enwiki_on_Jun_26_2019.png" class="image"><img alt="Screenshot delete log of enwiki on Jun 26 2019.png" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/9e/Screenshot_delete_log_of_enwiki_on_Jun_26_2019.png/1000px-Screenshot_delete_log_of_enwiki_on_Jun_26_2019.png" decoding="async" width="1000" height="865" srcset="//upload.wikimedia.org/wikipedia/commons/9/9e/Screenshot_delete_log_of_enwiki_on_Jun_26_2019.png 1.5x" data-file-width="1166" data-file-height="1009"></a> </p> 
      <ol>
       <li>The type of log to display. By default, all public logs will be shown, except the <a href="/wiki/Wikipedia:New_pages_patrol/patrolled_pages" title="Wikipedia:New pages patrol/patrolled pages">patrol</a>, <a href="/wiki/Wikipedia:Pending_changes" title="Wikipedia:Pending changes">review</a>, <a href="/wiki/Wikipedia:Tags" title="Wikipedia:Tags">tag</a> and <a href="/wiki/Help:Notifications/Thanks" title="Help:Notifications/Thanks">thanks</a> logs. They can be enabled at "Show additional logs" when "All public logs" is selected. The example is set to only show the deletion log, which lists page <a href="/wiki/Help:Delete" class="mw-redirect" title="Help:Delete">deletions and undeletions</a>.</li> 
       <li>The <b>Performer</b> field lets you search the logs for entries that were performed <i>by</i> a specified user. Type the username of the user you want to look up the logs for.</li> 
       <li>The <b>Target</b> field lets you search the logs for entries that were performed <i>on</i> a page or user. Type the name of the page you want to look up the logs for. If you want to look up the logs performed <i>on</i> a user, add <b>User:</b> (note the colon) to the start of your search query, then type in the username.</li> 
       <li>The <b>From date</b> field can be used to show only the logs created on or before a specified date.</li> 
       <li>The <b>Tag filter</b> allows you to show only logs that have been <a href="/wiki/Wikipedia:Tags" title="Wikipedia:Tags">tagged</a> with the specified name of the tag.</li> 
       <li>Some logs will have additional filtering options to show only specific kinds of actions. In this case, the deletion log can be further filtered to show only page deletions, page undeletions, <a href="/wiki/Wikipedia:Revision_deletion" title="Wikipedia:Revision deletion">revision deletions</a>, and the overwriting of <a href="/wiki/Help:Redirect" title="Help:Redirect">redirects</a>.</li> 
       <li>The <i>newest / oldest</i> links let you quickly jump to the newest log entries or to the oldest log entries. The <i>newer n / older n</i> links take you to the next or previous page of entries. The blue numbers list the number of entries displayed on the page. You can choose to display 20, 50, 100, 250 or 500 entries at once.</li> 
       <li>The date and time of the logged action. The time is <a href="/wiki/UTC" class="mw-redirect" title="UTC">UTC</a> by default on Wikipedia; registered users can change this in their <a href="/wiki/Special:Preferences" title="Special:Preferences">preferences</a>.</li> 
       <li>The <a href="/wiki/Help:Username" class="mw-redirect" title="Help:Username">username</a> or <a href="/wiki/IP_address" title="IP address">IP address</a> of the user who performed the action. The <i>talk</i> link points to their <a href="/wiki/Help:Talk_page#User_talk_pages" class="mw-redirect" title="Help:Talk page">user talk page</a>, while the <i>contribs</i> link points to their <a href="/wiki/Help:User_contributions" title="Help:User contributions">user contributions</a> page.</li> 
       <li>The type of action performed.</li> 
       <li>The name of the page or user that the action was performed on.</li> 
       <li>When performing most actions, the user will be prompted to provide an optional summary or explanation of their actions, which will be shown here.</li> 
       <li>The <a href="/wiki/Wikipedia:Tags" title="Wikipedia:Tags">tags</a> attached to the action, if applicable. Tags are usually automatically added to actions and are used for maintenance or tracking purposes.</li>
      </ol> 
      <h2><span class="mw-headline" id="Other_sources_of_logged_information">Other sources of logged information</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Help:Log&amp;action=edit&amp;section=3" title="Edit section: Other sources of logged information">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>Log files which are not automatically generated can be found in <a href="/wiki/Category:Wikipedia_logs" title="Category:Wikipedia logs">Category:Wikipedia logs</a>. Old log files of actions which have now switched to automatic logging can be found in <a href="/wiki/Category:Wikipedia_obsolete_log_pages" title="Category:Wikipedia obsolete log pages">Category:Wikipedia obsolete log pages</a>. Some of these old files are listed by type at <a href="/wiki/Wikipedia:Block_log" title="Wikipedia:Block log">WP:Block log</a>, <a href="/wiki/Wikipedia:Bureaucrat_log" title="Wikipedia:Bureaucrat log">WP:Bureaucrat log</a>, <a href="/wiki/Wikipedia:Deletion_log" title="Wikipedia:Deletion log">WP:Deletion log</a>, <a href="/wiki/Wikipedia:Protection_log" title="Wikipedia:Protection log">WP:Protection log</a> and <a href="/wiki/Wikipedia:Upload_log" title="Wikipedia:Upload log">WP:Upload log</a>. (These actions moved to automatic logging in December 2004.) </p>
      <p>Several other pages in the "Special:" namespace (such as <a href="/wiki/Special:RecentChanges" title="Special:RecentChanges">Special:RecentChanges</a>) serve similar functions to those of logs. For these, see <a href="/wiki/Help:Special_pages" class="mw-redirect" title="Help:Special pages">Help:Special pages</a>. </p>
      <p>Still more information is provided by <a href="/wiki/Wikipedia:Database_reports" title="Wikipedia:Database reports">Wikipedia:Database reports</a>. For example, <a href="/wiki/Wikipedia:Database_reports/Users_by_log_action" class="mw-redirect" title="Wikipedia:Database reports/Users by log action">Wikipedia:Database reports/Users by log action</a> contains a list of users ordered by number of logged actions performed. </p>
