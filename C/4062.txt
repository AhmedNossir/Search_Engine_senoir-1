        <h2><span class="mw-headline" id="Welcome_to_the_Wikimedia_Commons_community_portal">Welcome to the Wikimedia Commons community portal</span></h2> 
        <div class="unregisteredonly">
         <span style="margin-left: 3em;">
          <div class="floatright">
           <a href="/wiki/Special:UserLogin/signup" title="Special:UserLogin/signup"><img alt="MW-StyleGuide-CallToAction.png" src="https://upload.wikimedia.org/wikipedia/commons/1/10/MW-StyleGuide-CallToAction.png" decoding="async" width="215" height="75" data-file-width="215" data-file-height="75"></a>
          </div></span>
        </div> 
        <p>Are you new to Wikimedia Commons? If so, then <a href="/wiki/Commons:Welcome" title="Commons:Welcome">Welcome</a>! Check out <a href="/wiki/Commons:Project_scope" title="Commons:Project scope">what we're all about</a> (and <a href="/wiki/Commons:What_Commons_is_not" title="Commons:What Commons is not">what we are <i>not</i></a>), or just start <a href="/wiki/Special:Search" title="Special:Search">searching</a> for images and other media files. You can find collections of similar files grouped <a href="/wiki/Category:Topics" title="Category:Topics">by topic</a>, or <a href="/wiki/Category:Categories" title="Category:Categories">by almost any other characteristic</a> you can think of. You can even <a href="/wiki/Commons:First_steps" title="Commons:First steps">start contributing to the project</a> in important ways, regardless of your <a href="/wiki/Category:Users_by_ability" title="Category:Users by ability">abilities</a> or <a href="/wiki/Category:Users_by_interest" title="Category:Users by interest">interests</a>, and no matter <a href="/wiki/Category:Users_by_location" title="Category:Users by location">where you live</a> or <a href="/wiki/Category:Users_by_language" title="Category:Users by language">what language you speak</a>. All information about Commons itself is available <a href="/wiki/Category:Commons-en" title="Category:Commons-en">in English</a>, and most of it is available <a href="/wiki/Category:Commons_by_language" title="Category:Commons by language">in other languages</a>, as well. </p>

         </div> 
         <div style="padding: 0.4em 1em 0.3em 1em;"> 
          <dl>
           <dt>
            <a href="/wiki/Help:Contents" title="Help:Contents">Help index</a>
           </dt> 
           <dd>
            All help, project, maintenance, discussion, policy pages and further tutorials about how to work in wikis like Wikimedia Commons are listed there. For an overview over the structure of the Wikimedia Commons project itself see <a href="/wiki/Category:Commons" title="Category:Commons">Category:Commons</a>.
           </dd> 
           <dt>
            <a href="/wiki/Commons:Software" title="Commons:Software">Software</a>
           </dt> 
           <dd>
            Recommended software for each media type in order to create and edit the content you want to share at Wikimedia Commons.
           </dd> 
           <dt>
            <a href="/wiki/Commons:Tools" title="Commons:Tools">Tools</a>
           </dt> 
           <dd>
            Tools for uploading and further maintenance of Wikimedia Commons as for example <a href="/wiki/Commons:Tools/Commonist" class="mw-redirect" title="Commons:Tools/Commonist">Commonist</a> for uploading large numbers of images. <a href="/wiki/Commons:Tool_integration" class="mw-redirect" title="Commons:Tool integration">Tool integration</a> helps you add functionality to your Wikimedia Commons web interface.
           </dd> 
           <dt>
            <a href="/wiki/Commons:First_steps/Reuse" title="Commons:First steps/Reuse">How to use Wikimedia Commons' content</a>
           </dt> 
           <dd>
            Overview on how to embed Wikimedia Commons content into various Wikimedia wikis and how to reuse Commons' content by third parties. See also the <a href="/wiki/Commons:Media_help" title="Commons:Media help">media help</a> for software and hints how to watch images and films and listen to audio files provided by Wikimedia Commons. See <a href="/wiki/Commons:Featured_pictures" title="Commons:Featured pictures">Featured pictures</a>, <a href="/wiki/Commons:Valued_images" title="Commons:Valued images">Valued images</a>, <a href="/wiki/Commons:Picture_of_the_day" title="Commons:Picture of the day">Picture of the day</a> and <a href="/wiki/Commons:Media_of_the_day" title="Commons:Media of the day">Media of the day</a> for particularly valuable content you can reuse.
           </dd>

         </div>
         <div style="padding: 0.4em 1em 0.3em 1em;"> 
          <dl>
           <dt>
            <a href="/wiki/Commons:Project_scope" title="Commons:Project scope">Wikimedia Commons project scope</a>
           </dt> 
           <dd>
            The project definition and its derived detailed policy pages on <a href="/wiki/Commons:Licensing" title="Commons:Licensing">licensing</a>, allowed <a href="/wiki/Commons:Copyright_tags" title="Commons:Copyright tags">copyright tags</a> and <a href="/wiki/Commons:File_types" title="Commons:File types">file types</a> give you important information what we are and what not.
           </dd> 
           <dt>
            <a href="/wiki/Commons:Deletion_policy" title="Commons:Deletion policy">Deletion policy</a>
           </dt> 
           <dd>
            The deletion policy is closely related to the Wikimedia Commons project scope. It defines when and how to delete content that does not belong on Wikimedia Commons for various reasons.
           </dd> 
           <dt>
            <a href="/wiki/Commons:Language_policy" title="Commons:Language policy">Multilingualism</a>
           </dt> 
           <dd>
            Wikimedia Commons is multilingual. Use <a href="/wiki/Commons:Babel" title="Commons:Babel">Babel</a> in order to find contributors who speak your language. Please add information about the languages you understand to your user page. The language policy is discussed at <a href="/wiki/Commons_talk:Language_policy" title="Commons talk:Language policy">Commons talk:Language policy</a>.
           </dd>
