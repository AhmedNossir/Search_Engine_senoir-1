           <h2>Présentation</h2>
           <p>Le tableau de bord Cloudflare gère les configurations de compte et de domaine. Les fonctionnalités principales du tableau de bord sont&nbsp;les suivantes&nbsp;:</p>
           <ul>
            <li>Contrôler les fonctionnalités Cloudflare de contrôle,</li>
            <li>Obtenir des informations sur sécurité du domaine et l’analyse de mise en cache,</li>
            <li><a href="https://support.cloudflare.com/hc/articles/200167946">Partager l’accès à un compte</a>avec des utilisateurs supplémentaires, et</li>
            <li>Configurer vos paramètres à partir de n’importe quel appareil grâce à la conception flexible de notre tableau de bord.</li>
           </ul>
           <p>Toutes les offres Cloudflare permettent de <a href="https://support.cloudflare.com/hc/articles/200167946">partager l’accès à un compte avec utilisateurs supplémentaires</a>. Par ailleurs, les offres Enterprise permettent aux Administrateurs de créer des <a href="https://support.cloudflare.com/hc/articles/205065067">comptes multi-utilisateurs</a>, ce qui permet à chaque utilisateur de modifier les paramètres Cloudflare en fonction de leur rôle.</p>
           <div class="cf-sidebar-2 cf-note">
            Le tableau de bord Cloudflare nécessite un navigateur web compatible avec <a href="https://caniuse.com/#search=tls%201.2" target="_self">TLS&nbsp;1.2</a> ou une version plus récente. Le tableau de bord Cloudflare n’est pas pris en charge sur les versions plus anciennes que le navigateur IE 10.&nbsp;&nbsp;Consultez <a href="https://browsehappy.com/?locale=en">Browse Happy</a> pour obtenir les liens de mise à niveau des navigateurs les plus courants.
           </div>
           <p>Cloudflare utilise des jetons pour authentifier une session lorsqu’un utilisateur est connecté à son compte Cloudflare. Un message de jeton invalide ou un message d’erreur par e-mail apparaît si votre FAI change votre adresse IP pendant la session, du fait que ce changement invalide le jeton. Déconnectez-vous puis reconnectez-vous pour corriger le problème ou videz le cache de votre navigateur ainsi que les cookies.</p>
           <hr>
           <h2>Résumé des applications du tableau de bord Cloudflare</h2>
           <p>Le tableau de bord Cloudflare contient plusieurs applications destinées à la <a href="#h_76770eb2-b211-4e55-826f-8111c1d2cf30" target="_self">connaissance approfondie des données</a>, <a href="#h_fd2d3e1b-c379-4a41-bfb0-b643f9fd127a" target="_self">l’optimisation de la sécurité</a>, <a href="#h_6f753407-f117-4162-9bdc-0252f09884a6" target="_self">une flexibilité de la configuration</a>, <a href="#h_7764d480-95df-4aa8-8eeb-12a9e17079e9" target="_self">l’amélioration de la performance</a> ainsi qu’une fiabilité accrue.</p>
           <p><img alt="dashboard_apps_no_crypto_Sept19.png" src="https://support.cloudflare.com/hc/article_attachments/360038338392/dashboard_apps_no_crypto_Sept19.png"></p>
           <p>Les informations ci-dessous renseignent la fonction principale de chaque application du tableau de bord&nbsp;:</p>
           <h3 id="h_76770eb2-b211-4e55-826f-8111c1d2cf30">Connaissance approfondie&nbsp;</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;Présentation</strong></p>
           <p class="cf-video-container"><iframe allowfullscreen height="320" src="https://iframe.cloudflarestream.com/1396ff01893962ed4e4471507b428212" style="border: none;" width="480"></iframe></p>
           <p>&nbsp; &nbsp; &nbsp;Apporte les fonctionnalités suivantes&nbsp;:</p>
           <ul>
            <li>Synthétise les statistiques du site</li>
            <li>Affiche les notifications</li>
            <li>Liens vers les actions rapides communes</li>
            <li>Résume les extensions d’offre et les fonctionnalités supplémentaires</li>
            <li>Apporte les détails d’authentification API, dont <strong>l’ID de zone</strong> et <strong>l’ID de compteD</strong></li>
            <li>Permet de changer les offres de domaine</li>
            <li>Permet de supprimer un site de Cloudflare&nbsp;</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Statistiques</strong></p>
           <p>&nbsp; &nbsp; &nbsp;Surveille les statistiques et les tendances des points suivants&nbsp;:</p>
           <ul>
            <li>Trafic visiteur</li>
            <li>Gestion des requêtes DNS</li>
            <li>Détection de menace</li>
            <li>Performance</li>
           </ul>
           <h3 id="h_fd2d3e1b-c379-4a41-bfb0-b643f9fd127a">Sécurité</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;Spectrum</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://developers.cloudflare.com/spectrum/getting-started/">Protège tous les ports TCP/UDP</a>&nbsp;de la couche 3 et 4 <a href="https://www.cloudflare.com/learning/ddos/what-is-a-ddos-attack/">Attaques de déni de service</a></li>
            <li>Arrêter l’espionnage de trafic en activant le chiffrement TLS pour les services TCP</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;SSL/TLS</strong>&nbsp;:</p>
           <ul>
            <li>Gérer <a href="https://support.cloudflare.com/hc/en-us/articles/203295200" target="_self">les produits de certificat SSL Cloudflare</a></li>
            <li>Ajuster<a href="https://support.cloudflare.com/hc/articles/203041594#h_812cc429-890f-421e-9c29-5fb895157522">la compatibilité du navigateur</a>&nbsp;en modifiant les paramètres TLS et les paramètres de chiffrement</li>
            <li><a href="https://support.cloudflare.com/hc/articles/204144518#h_a61bfdef-08dd-40f8-8888-7edd8e40d156">Rediriger toutes les requêtes HTTP pour sécuriser HTTPS</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Pare-feu</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/217074967">Bloquer ou autoriser le trafic&nbsp;</a> basé sur l’adresse IP, la plage d’adresses IP, le pays ou l’agent utilisateur </li>
            <li><a href="https://support.cloudflare.com/hc/articles/235240767">Fixe des limites de taux</a>&nbsp;et limite l’accès à des URL spécifiques pour des IP spécifiques</li>
            <li>Ajuste les paramètres <a href="https://support.cloudflare.com/hc/articles/360002866492">OWASP</a> et <a href="https://support.cloudflare.com/hc/articles/200172016">de règle WAF</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200170086">Évalue les en-têtes HTTP du visiteurs pour les menaces</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Access</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://developers.cloudflare.com/access/about/">Élimine les VPN sans que cela n’exige de changement au niveau de votre serveur d’origine </a></li>
            <li>Protège les ressources internes en exigeant une authentification</li>
            <li>Contrôle quel utilisateur ou groupe peut accéder à des ressources sensibles</li>
            <li>Prend connaissance des journaux d’accès</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;ScrapeShield</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/200170036">Protège</a> les informations sensibles des spammeurs et des bots</li>
            <li><a href="https://support.cloudflare.com/hc/articles/200170016">Rend les adresses électroniques opaques</a>&nbsp;pour empêcher le bots de scraper</li>
            <li><a href="https://support.cloudflare.com/hc/articles/200170026">Empêche l’abus des liens dynamiques d’images</a>&nbsp;par des sites non approuvés</li>
           </ul>
           <h3 id="h_6f753407-f117-4162-9bdc-0252f09884a6">Flexibilité</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;Workers</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://developers.cloudflare.com/workers/about/">Exécute les applications JavaScript dans un environnement d’exécution sans serveur</a></li>
            <li><span>Échelonne facilement les applications sur l’infrastructure ou les opérations</span></li>
            <li><span>Construit un contrôle granulaire des requêtes et des réponses</span></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Page</strong> <strong>Rules</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/218411427">Offre un contrôle granulaire</a>&nbsp;pour beaucoup de nombreux paramètres Cloudflare en fonction d’une URL correspondante</li>
            <li><a href="https://support.cloudflare.com/hc/articles/360021023712">Mettre en cache du contenu HTML statique</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Custom</strong> <strong>Pages</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/200172706">Crée des pages d’erreur personnalisées</a>&nbsp;pour les blocs IP, les blocs WAF, les erreurs HTTP 5XX, les erreurs HTTP 1XXX Cloudflare et plus encore</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Applications</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/115000305752">Visualise et installe</a>&nbsp;une large gamme d’applications qui améliorent la sécurité, la performance, les statistiques, le design, etc. de votre site</li>
           </ul>
           <h3 id="h_7764d480-95df-4aa8-8eeb-12a9e17079e9">Performance</h3>
           <p><strong>&nbsp; &nbsp; &nbsp;DNS</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/360019093151">G</a><a href="https://support.cloudflare.com/hc/articles/360019093151">ère les registres DNS du domaine</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200169626" target="_self">Choisit quels enregistrements A ou CNAME proxyser</a>&nbsp;via Cloudflare</li>
            <li>Trouve les serveurs de noms Cloudflare attribués à votre domaine</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Speed</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/219178057">Accélère le temps de chargement sur les appareils mobiles</a>&nbsp;qui utilisent des réseaux lents</li>
            <li><a href="https://support.cloudflare.com/hc/articles/360000607372">Compresse les images pour accélérer le chargement de pages </a></li>
            <li>Accélère la vitesse du site en<a href="https://support.cloudflare.com/hc/articles/200168196" target="_self">minimisant la taille de votre code source d’origine</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200168056">Reporte le chargement de JavaScript</a>&nbsp;jusqu’au rendu de votre site</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Mise en cache</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/200168256">Ajuste le niveau de mise en cache</a></li>
            <li><a href="https://support.cloudflare.com/hc/articles/200169246">Efface les fichiers mis en cache</a>&nbsp;de tous les datacenters Cloudflare</li>
            <li><a href="https://support.cloudflare.com/hc/articles/202775670">Indique à Cloudflare ce qu’il faut mettre en cache</a>&nbsp;et <a href="https://support.cloudflare.com/hc/articles/200168276">définit l’expiration du cache du navigateur</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; Réseau</strong>&nbsp;:</p>
           <ul>
            <li>Fonctionnalités de choix permettant d’améliorer les performances réseau comme <a href="https://www.cloudflare.com/website-optimization/http2/what-is-http2/">HTTP/2</a>&nbsp;et <a href="https://cloudflare-quic.com/">QUIC</a></li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Trafic</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://www.cloudflare.com/products/argo-smart-routing/">Réduit la latence et les erreurs de connexion</a></li>
            <li><a href="https://developers.cloudflare.com/argo-tunnel/quickstart/">P</a><a href="https://developers.cloudflare.com/argo-tunnel/quickstart/">rotège les serveurs web d’origine contre l’exposition et les attaques</a>&nbsp;d’adresses IP</li>
            <li><a href="https://support.cloudflare.com/hc/articles/115000081911">Équilibre le trafic</a>&nbsp;et empêche les interruptions</li>
           </ul>
           <p><strong>&nbsp; &nbsp; &nbsp;Stream</strong>&nbsp;:</p>
           <ul>
            <li><a href="https://support.cloudflare.com/hc/articles/360017801091">Encode, stocke et diffuse vos vidéos</a></li>
            <li>Optimise automatiquement le format et le débit binaire pour chaque appareil et réseau</li>
           </ul>
