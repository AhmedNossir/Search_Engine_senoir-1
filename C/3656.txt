   <section id="faq" class="section" aria-label="Frequently asked questions about Azure Bot Service" data-module="faq-01"> 
    <div class="row"> 
     <div class="column medium-3"> 
      <h2>Frequently asked questions about Azure Bot Service</h2> 
     </div> 
     <div class="column medium-8 medium-offset-1"> 
      <ul class="faq"> 
       <li class="faq__item"> <button type="button" class="faq__question" aria-expanded="false" aria-controls="481d18bd" data-bi-slot="1" data-bi-id="faq-1" data-bi-an="faq-01"> Where should I start? </button> 
        <div class="faq__answer" id="481d18bd">
         <p><a href="https://docs.microsoft.com/en-us/azure/bot-service/bot-service-overview-introduction?view=azure-bot-service-4.0">Bot Framework</a> is a comprehensive conversational AI development environment for designing conversational AI experiences. <a href="https://docs.microsoft.com/en-us/composer/introduction">Bot Framework Composer</a>, <a href="https://docs.microsoft.com/en-us/azure/bot-service/?view=azure-bot-service-4.0">Azure Bot Service</a>, <a href="https://docs.microsoft.com/en-us/azure/health-bot/overview">Azure Health Bot</a>, and <a href="https://docs.microsoft.com/en-us/power-virtual-agents/fundamentals-what-is-power-virtual-agents">Power Virtual Agents</a> are built on Bot Framework.</p> 
         <p><a href="https://docs.microsoft.com/en-us/azure/cognitive-services/qnamaker/overview/overview">QnA Maker</a> is an Azure Cognitive Service that automatically extracts question-and- answer pairs from semi-structured documents, including website FAQ. QnA Maker makes it easy to create simple bots without the need to write code.</p> 
         <p><a href="https://docs.microsoft.com/en-us/azure/bot-service/?view=azure-bot-service-4.0">Azure Health Bot</a> is specifically designed to build bots for the healthcare domain. <a href="https://docs.microsoft.com/en-us/power-virtual-agents/fundamentals-what-is-power-virtual-agents">Power Virtual Agents</a> is a no-code bot-building experience, available as a software as a service (SaaS) on <a href="https://powerplatform.microsoft.com/en-us/what-is-power-platform/">Microsoft Power Platform</a>.</p>
        </div> </li> 
       <li class="faq__item"> <button type="button" class="faq__question" aria-expanded="false" aria-controls="7e722593" data-bi-slot="2" data-bi-id="faq-2" data-bi-an="faq-01"> What is Bot Framework Composer? </button> 
        <div class="faq__answer" id="7e722593">
         <p><a href="https://docs.microsoft.com/en-us/composer/introduction">Bot Framework Composer</a> is a visual, open-source bot authoring canvas to build, test, and deploy bots directly to Azure. Start with a simple question-and-answer bot extracted from semi-structured documents or a website, or with prebuilt components that can be extended with code and logic to create sophisticated multilingual and multimodal bots for any scenario.</p> 
         <p>Bots built in Power Virtual Agents can be extended in Bot Framework Composer.</p>
        </div> </li> 
       <li class="faq__item"> <button type="button" class="faq__question" aria-expanded="false" aria-controls="d529d0b5" data-bi-slot="3" data-bi-id="faq-3" data-bi-an="faq-01"> Can I use open-source software with Bot Framework? </button> 
        <div class="faq__answer" id="d529d0b5">
         Yes. The Bot Framework SDKs and tools are all built as open-source technologies, available through the <a href="https://github.com/microsoft/botframework-sdk">GitHub repo</a>. In addition, you can integrate your own components and technologies in your bot solution as needed for your scenario.
        </div> </li> 
       <li class="faq__item"> <button type="button" class="faq__question" aria-expanded="false" aria-controls="9cf894ec" data-bi-slot="4" data-bi-id="faq-4" data-bi-an="faq-01"> Does Bot Framework provide analytics for my bot? </button> 
        <div class="faq__answer" id="9cf894ec">
         Yes. As a developer solution, Bot Framework provides designs that utilize <a href="/en-us/services/monitor/">Application Insights</a> to collect service-level and conversation-level data that can help gauge the performance and efficacy of your bot. Application Insights is available for bots built directly with the <a href="https://docs.microsoft.com/en-us/azure/bot-service/bot-service-overview-introduction?view=azure-bot-service-4.0">Bot Framework SDK</a>, as well as <a href="https://docs.microsoft.com/en-us/composer/introduction">Bot Framework Composer</a>.
        </div> </li> 
       <li class="faq__item"> <button type="button" class="faq__question" aria-expanded="false" aria-controls="f06a448a" data-bi-slot="5" data-bi-id="faq-5" data-bi-an="faq-01"> Does Bot Framework include natural language AI and speech? </button> 
        <div class="faq__answer" id="f06a448a">
         Yes. Bot Framework uses <a href="/en-us/services/cognitive-services/language-understanding-intelligent-service/">Language Understanding (LUIS)</a> for building custom machine-learning language models to extract information from conversational phrases and <a href="https://speech.microsoft.com/">Azure Speech</a> and <a href="https://docs.microsoft.com/en-us/azure/cognitive-services/speech-service/direct-line-speech">Direct Line Speech</a> for creating intuitive and extensible voice-based interaction with bots.
        </div> </li> 
       <li class="faq__item"> <button type="button" class="faq__question" aria-expanded="false" aria-controls="232141fa" data-bi-slot="6" data-bi-id="faq-6" data-bi-an="faq-01"> Which languages are supported by Bot Framework? </button> 
        <div class="faq__answer" id="232141fa">
         <p>The Bot Framework SDK and Bot Framework Composer provide <a href="https://docs.microsoft.com/en-us/composer/how-to-use-multiple-language">multilingual support</a> for bot development for a variety of languages, with English as the default.</p> 
         <p>The Bot Framework SDK supports multiple <a href="https://github.com/microsoft/botframework-sdk/#readme">programming languages</a>, including C#, JS, Python, and Java.</p> 
         <p><a href="https://docs.microsoft.com/en-us/composer/introduction">Bot Framework Composer</a> contains reusable assets in the form of JSON and Markdown files that can be bundled and packaged with a bot's source code, which can be checked into source control systems and deployed along with code updates such as dialogs, language understanding training data, and message templates.</p>
        </div> </li> 
      </ul> 
     </div> 
    </div>
