      <h3>More work</h3> 
      <p>I manage the website of the <a href="http://www.newsargus.com">Goldsboro News-Argus</a>, a local newspaper. The current design of the site is mostly my work, but has been heavily affected by decisions and requests of others. </p> 
      <p>Where the visual design may be a bit lacking to my taste, I have been able to focus on the site’s accessibility, and proper use of web standards and semantics. The following techniques are used:</p> 
      <ul> 
       <li>CSS sprites – to minimize HTTP requests, allowing the site to load faster</li> 
       <li>Unobtrusive JavaScript – To maintain functionality with those without JavaScript, as well as keeping the HTML clean and lean</li> 
       <li>An HTML/CSS design that allows the page content to load before the sidebar(s)</li> 
       <li>8-bit PNGs with alpha transparency to keep image sizes low</li> 
       <li>Pure-CSS drop-down lists – To keep the links accessible</li> 
       <li>Minified JavaScript and CSS files to keep download speed fast</li> 
       <li>CSS visual enhancements where supported</li> 
       <li>JavaScript interaction enhancement (using jQuery) where supported</li> 
       <li>CSS rules for older IE versions kept separate using conditional style sheets</li> 
       <li>A semi-liquid design is used to optimize screen usage and minimize scrolling</li> 
      </ul>
