      <p>Following is a list of Wikimedia Commons shortcuts for English pages (<small><a href="/wiki/Special:PrefixIndex/COM:" title="Special:PrefixIndex/COM:">Commons</a> — <a href="/wiki/Special:PrefixIndex/CAT:" title="Special:PrefixIndex/CAT:">Categories</a></small>). <kbd>CAT:</kbd> is an alias for <kbd>Category:</kbd> (see <a href="/wiki/Help:Namespaces" title="Help:Namespaces">Help:Namespaces</a>), <kbd>CT:</kbd> is a not (yet) very popular prefix for shortcuts to important Commons talk pages. <kbd>COM:</kbd> is actually an alias for <kbd>Commons:</kbd> or <kbd>Project:</kbd>, e.g., <a href="/wiki/Commons:COM" class="mw-redirect" title="Commons:COM">COM:COM</a> and <a href="/wiki/Commons:COM" class="mw-redirect" title="Commons:COM">Project:COM</a> arrive on <a href="/wiki/Commons:COM" class="mw-redirect" title="Commons:COM">Commons:COM</a>, where they are redirected to this page <a class="mw-selflink selflink">Commons:Shortcuts</a>. Likewise <a class="mw-selflink selflink">COM:Shortcuts</a> and <a class="mw-selflink selflink">Project:Shortcuts</a> go to this page (without redirection). </p>
